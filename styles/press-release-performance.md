# 스타일 가이드: 실적/성과 보도자료 (Performance Press Release)

> 메이아이의 매출 성장, 고객사 확대, 신규 서비스 출시 등 사업 성과를 다루는 보도자료의 작성 스타일 가이드.
> 분석 대상: AI타임스, ZDNet Korea, K스타트업밸리, 아시아경제, 헬로티, 뉴스탭, 디스커버리뉴스 등 보도 기사 10편

---

## 1. 문체

### 어미 패턴
- **기본 어미:** `~다` 체 (경어체가 아닌 서술체)
- 주로 사용되는 어미:
  - `~밝혔다`, `~발표했다` (발표/공시 사실 전달)
  - `~증가했다`, `~늘었다`, `~올랐다` (수치 변화)
  - `~달한다`, `~달성했다` (성과 강조)
  - `~확보했다`, `~거뒀다` (실적 결과)
  - `~이다`, `~솔루션이다` (정의/설명)
  - `~중이다`, `~분석 중이다` (현재 진행)
  - `~방침이다`, `~계획이다`, `~예정이다` (향후 계획)

### 격식 수준
- 뉴스 보도문 스타일의 **객관적 서술체**
- 감탄이나 주관적 평가 배제
- `~입니다`(합쇼체)는 사용하지 않음

### 실제 예시
```
메이아이(대표 김찬규)가 2025년 상반기 매출이 전년 동기 대비 130% 이상 증가했다고 22일 발표했다.
```
```
현재 AI 엔진은 하루 평균 2만 5,000시간 이상의 영상 데이터를 실시간 분석 중이다.
```
```
글로벌 박람회 및 IR 프로그램을 통해 해외 접점을 넓히고, 매쉬의 글로벌 상용화에 집중할 방침이다.
```

## 2. 글 구조

### 전형적 구조 (5~7단락)

```
[리드] 핵심 실적 수치 + 발표 사실
[본문1] 매쉬 솔루션 소개/설명
[본문2] 성장 요인 (3가지 항목 나열)
[본문3] 주요 고객사 및 적용 사례
[본문4] 기술력 입증 (학회, 박람회 등) 또는 신규 서비스
[본문5] 향후 계획 (글로벌 확장)
[마무리] 대표 인용문
```

### 구조 특징
- 리드 → 제품 설명 → 성과 상세 → 미래 계획 → 대표 코멘트 순서가 고정적
- 소제목(###)을 활용해 섹션을 구분하는 매체도 있으나 필수는 아님
- 마지막 단락은 항상 대표 발언(인용문)으로 마무리

## 3. 리드(첫 문단) 작성법

### 패턴
**[회사 수식어] + [회사명(대표명)] + [핵심 실적 수치] + [발표 동사]**

### 정보 배치 순서
1. 회사 수식어 (예: "영상처리 AI 스타트업", "영상처리 AI 전문기업")
2. 회사명 + 대표명 (예: "메이아이(대표 김찬규)")
3. 핵심 실적 (예: "2025년 상반기 매출이 전년 동기 대비 130% 이상 증가했다")
4. 발표 동사 + 날짜 (예: "~고 22일 밝혔다", "~고 발표했다")

### 리드 실제 예시
```
영상처리 AI 스타트업 메이아이(대표 김찬규)가 2025년 상반기 매출이 전년 동기 대비 130% 이상 증가했다고 22일 발표했다.
```
```
영상처리 AI 전문기업 메이아이(대표 김찬규)가 2025년 상반기 매출이 전년 동기 대비 130% 이상 늘었다고 발표했다.
```
```
영상처리 AI 전문 기업 메이아이(대표 박준혁)는 2019년 창사 이후 매년 두 배 이상의 매출 증가율을 달성하고 있다고 밝혔다.
```
```
메이아이가 AI 기반 오프라인 매장 분석 시장에서의 입지를 실적 성장으로 입증했다.
```

### 리드 변형
- **수식어 생략형:** "메이아이가 2025년 상반기 기준 전년 동기 대비 매출이 130% 이상 증가했다고 22일 밝혔다."
- **성과 강조형:** "메이아이가 AI 기반 오프라인 매장 분석 시장에서의 입지를 실적 성장으로 입증했다." (수치는 다음 문단으로)

## 4. 자주 쓰는 표현/관용구

### 성과 표현
- `전년 동기 대비 130% 이상 증가`
- `연평균 200%의 매출 성장을 달성`
- `성장세를 이어가고 있다`
- `적용 사례를 빠르게 늘려가고 있다`
- `입지를 실적 성장으로 입증했다`

### 발표/공시 동사
- `~라고 밝혔다` / `~라고 발표했다` / `~라고 말했다`
- `~라는 포부를 밝혔다`
- `~라는 의지를 밝혔다`

### 성장 요인 나열
- `▲차별화된 기술력 ▲효율적인 운영 역량 ▲팝업 플랜 출시를 꼽았다`
- 불릿(-)이나 번호(1, 2, 3) 목록으로 3가지 요인 나열이 전형적

### 기술/제품 수식어
- `자체 개발 재식별화 기술`
- `딥러닝 기반 알고리즘`
- `AI 기반 매장 분석 솔루션`
- `CCTV 영상을 AI로 분석해`
- `실질적 인사이트를 제공`
- `정량적으로 파악`

### 계획/전망 표현
- `~에 집중할 방침이다`
- `~에 집중할 계획이다`
- `~을 추진할 계획이다`
- `글로벌 시장 공략에도 속도를 낼 계획이다`
- `글로벌 시장에서 인정받는 기업으로 성장하겠다`

### 반복 등장하는 문구 (거의 원문 그대로 재사용됨)
- `데이터 기반의 오프라인 공간 혁신을 이끌며`
- `글로벌 시장에서 인정받는 기업으로 성장하겠다`
- `기술 혁신과 시장 확장 측면에서 중요한 전환점`

## 5. 인용문 활용

### 배치 위치
- **항상 보도자료 맨 마지막 단락**에 위치
- 보도자료 중간에 인용문이 등장하는 경우는 없음

### 인용문 형식
**[이름] + [직책] + [회사명] + [대표는] + [인용 내용] + [발화 동사]**

### 발화 동사
- `~라고 밝혔다` (가장 빈번)
- `~라는 포부를 밝혔다`
- `~라는 의지를 밝혔다`
- `~라고 말했다`
- `~라고 전했다`

### 인용문 실제 예시
```
김찬규 메이아이 대표는 "비바테크 참여와 ICLR 논문 채택으로 글로벌 기술력을 인정받았고, 국내 주요 기업들과의 파트너십 확대로 시장에서의 입지를 더욱 확고히 했다"며, 데이터 기반의 오프라인 공간 혁신을 이끌며 글로벌 시장에서 인정받는 기업으로 성장하겠다는 포부를 밝혔다.
```
```
김찬규 대표는 "데이터 기반의 오프라인 공간 혁신을 이끌며 글로벌 시장에서 인정받는 기업으로 성장하겠다"고 밝혔다.
```
```
박준혁 대표는 "기존 CCTV 인프라를 활용한 공간 운영 해법과 팝업스토어 특화 솔루션으로 해외 진출 기반을 구축했다"며 2025년 글로벌 시장 확장 의지를 밝혔다.
```

### 인용문 구조 패턴
- **직접 인용 + 간접 인용 결합형:** `"직접 인용"며, ~라는 포부를 밝혔다`
- **직접 인용 단독형:** `"직접 인용"고 밝혔다`
- 인용문 내용은 **성과 요약 + 향후 비전**이 전형적

## 6. 숫자/데이터 표기법

### 매출/성장률
- `130% 이상 증가` (퍼센트는 아라비아 숫자 + %)
- `전년 동기 대비 130%` / `전년 대비 약 270%`
- `약`을 자주 붙임 (정확하지 않을 때)
- `연평균 200%의 매출 성장`

### 금액
- `최대 7억원` / `최대 7억 원` (원 앞 띄어쓰기 비통일)
- `12억 원 규모`

### 큰 숫자
- `1만 9,000시간` / `1만9000시간` / `1만9천 시간` (매체별 차이 존재)
- `2만 5,000시간 이상`
- `1,000만 시간` / `1천만 시간`

### 배수/비율
- `약 3배 증가` / `3배 이상 증가`
- `32% 증가`

### 날짜
- `2025년 상반기` / `지난해` / `올해`
- `22일 밝혔다` (발표일은 일(日)만 표기)
- `2024년부터`
- `지난 4월`

### 제목에서의 숫자 표기
- `130%↑` / `200%↑` (화살표 기호 사용)
- `전년比` (한자 약어)

## 7. 회사 소개 패턴

### 첫 등장 (리드)
- `영상처리 AI 스타트업 메이아이(대표 김찬규)`
- `영상처리 AI 전문기업 메이아이(대표 박준혁)`
- `영상처리 AI 전문 기업 메이아이(대표 박준혁)`
- `AI 기반 영상 분석 스타트업 ㈜메이아이(대표 김찬규)`

### 수식어 패턴
- **핵심 키워드:** `영상처리 AI` + `스타트업` 또는 `전문기업`
- 대표명은 괄호 안에 표기: `(대표 김찬규)`
- 법인 표기는 일부만 사용: `㈜메이아이`

### 제품(매쉬) 첫 소개
- `매장 분석 AI 솔루션 '매쉬(mAsh)'`
- `매장 내 CCTV 영상을 AI로 분석해 방문객의 연령대, 동선, 체류 시간 등 데이터를 측정`
- `KPI 등 실질적 인사이트를 제공해 공간 운영과 매장 경험 개선을 지원`

### 매쉬 소개 전형적 문장
```
매쉬는 매장 내 CCTV 영상을 AI로 분석해 방문객의 연령대, 동선, 체류 시간 등 데이터를 측정하고, 핵심성과지표(KPI)를 비롯한 실질적 인사이트를 제공해 공간 운영과 매장 경험 개선을 지원한다.
```

### 기술 차별점 소개
```
별도 장비 구매·시공 없이 기설치 CCTV에 즉시 도입 가능한 점이 강점이다.
```
```
기존 수기 조사나 WiFi 센서 기반 분석의 한계를 극복한 솔루션으로 평가받았다.
```

## 8. 문장 길이와 단락 구성

### 문장 길이
- 평균 **40~70자** 내외
- 짧은 문장: `현재 AI 엔진은 하루 평균 2만 5,000시간 이상의 영상 데이터를 실시간 분석 중이다.` (~40자)
- 긴 문장: `매쉬는 매장 내 CCTV 영상을 AI로 분석해 방문객의 연령대, 동선, 체류 시간 등 데이터를 측정하고, 핵심성과지표(KPI)를 비롯한 실질적 인사이트를 제공해 공간 운영과 매장 경험 개선을 지원한다.` (~90자)

### 단락당 문장 수
- **1~3문장**이 일반적
- 리드 단락: 1~2문장
- 본문 단락: 1~3문장
- 인용문 단락: 1문장

### 불릿/리스트 활용
- 성장 요인, 고객사, 기능 나열 시 `-` 불릿 또는 번호 목록 사용
- 3~5개 항목이 일반적

## 9. 전체 글 길이

- **짧은 보도자료:** 약 300~400자 (전자신문, 아시아경제 스타일)
- **중간 보도자료:** 약 500~700자 (AI타임스, ZDNet, 뉴스탭 스타일)
- **긴 보도자료:** 약 800~1,200자 (헬로티, K스타트업밸리, 디스커버리뉴스 스타일)
- **단락 수:** 5~8개 단락
- **소제목:** 긴 보도자료일수록 2~5개의 소제목(###) 사용

## 10. 제목 작성 패턴

### 형식
- `메이아이, [핵심 성과]...[인용/부연]`
- `[인용문]...메이아이, [성과 요약]`

### 실제 예시
```
메이아이, 연평균 매출 200%↑..."매장 성과 분석 AI 솔루션 '매쉬' 덕"
메이아이 "매장 성과 분석 AI '매쉬'로 연평균 매출 200% 성장"
메이아이, AI로 읽는 오프라인...'매쉬'로 상반기 매출 130%↑
"AI 분석 기술력 입증"…메이아이, 상반기 매출 130% 증가
메이아이, 상반기 매출 130% 증가…AI 솔루션 '매쉬' 견인
```

### 제목 특징
- 회사명 + 핵심 수치가 반드시 포함
- `↑` 기호로 증가 표현
- `...` 또는 `…`으로 부연 연결
- 솔루션명 '매쉬' 언급 빈도 높음
