# 아는 만큼 보인다! 방문객 데이터 분석 툴, 매쉬보드의 모든 것

**데이터 분석가가 만든 자체 개발 웹 대시보드, 매쉬보드를 소개합니다**

---

안녕하세요, 오프라인 공간을 더 똑똑하게 만드는 인공지능 스타트업 메이아이입니다.

메이아이는 인공지능 솔루션 '매쉬(mAsh)'로 매장과 방문객 데이터를 분석하고 도출합니다. 고객사의 매장 운영 전략을 좌지우지할 수 있는 정보를 취급하는 만큼 매쉬의 데이터를 고객사에게 소상히 보여드리는 것 또한 메이아이의 소명입니다.

메이아이는 매쉬의 데이터를 효과적으로 전달하기 위해 '매쉬보드(mAsh Board)'라는 이름의 웹 대시보드를 직접 개발하고 운영하고 있습니다. 그렇다면 매쉬보드는 구체적으로 어떤 데이터를 제공하며, 보통의 데이터 분석 툴과는 어떤 차이를 가질까요? **메이아이의 고객사만을 위해 태어난 데이터 분석 툴, 매쉬보드의 소개를 아래에서 자세히 전합니다.**

---

## 전문가가 관리하는 분석 툴, 매쉬보드

매쉬는 매장에 설치된 CCTV 영상을 통해 매장 방문객의 비정형적 데이터를 수집하고 분석합니다. 매쉬가 분석한 직후의 데이터는 통계의 형태를 갖추고 있지만, 아직까지 고객사가 조회하거나 사용하기 쉽지 않은 날 것에 가깝습니다. 따라서 매쉬가 도출한 데이터는 고객사에게 전달되기에 앞서 시각화 과정을 거쳐야 합니다. 이때 사용되는 것이 바로 메이아이가 개발한 웹 대시보드인 '매쉬보드'입니다.

매쉬가 도출한 데이터는 메이아이의 데이터 분석가가 전문적으로 관리합니다. 데이터는 매쉬보드 안에서 크게 차트와 대시보드 형태로 가공되며, 정제된 자료가 되었을 때 비로소 사용자 앞에 놓입니다. 다시 말해 매쉬보드는 고객사에게는 매장 운영의 인사이트를 발견하는 창구가 되고, 메이아이에게는 매쉬의 분석력과 정확도를 증명하는 무대가 된다고 할 수 있습니다.

데이터 기반의 의사결정을 처음 시작하는 마케터, 대규모 트래픽을 기반으로 인사이트를 도출해야 하는 데이터 분석가, 직감과 주관에서 벗어나 매장 운영의 합리성을 높이고 싶은 현장 관리자, 여러 매장의 성과를 통합 관리해야 하는 매장 매니저 등. **매쉬보드는 누구나 쉽게 사용하고 이해할 수 있는 서비스를 지향합니다.** 아래에 매쉬보드에서 확인할 수 있는 데이터의 종류와 제공 형태에 대한 설명을 붙입니다.

---

## 단순한 기능, 깔끔한 차트, 간편한 조회

매쉬보드는 매장 앞 유동 인구 수, 매장 방문객 수, 방문객의 성별과 연령대, 방문객의 동선, 방문객의 매장 총 체류 시간, 매장 내 특정 구역에 입장한 방문객 수와 그 구역별 체류 시간 등의 데이터를 제공합니다.

제공하는 데이터의 가짓수가 많은 만큼 데이터 분석이 처음이거나 매쉬보드의 숙련도가 낮은 사용자는 복잡한 조회 기능을 오히려 부담스럽게 느낄 수 있습니다. 그래서 매쉬보드는 **매장 운영의 필수 지표이자 사용자들이 가장 많이 조회하는 데이터를 간결한 인터페이스로 제공합니다.**

### 기본 통계 : 모든 매장 페이지

고객사의 모든 매장 정보를 한 번에 조회할 수 있는 페이지입니다.

### 기본 통계 : 매장 방문 페이지

매장 앞 유동 인구, 그중 매장을 방문한 사람의 수와 그 비율, 매장 방문객 중 구매가 일어난 비율 등. 방문객과 구매에 해당하는 핵심 KPI를 제공합니다. 고객사는 매장의 성과를 파악하고 병목을 찾아내어 개선하거나 주요 방문객의 정보를 파악하여 매장 구성 혹은 마케팅 전략을 최적화할 수 있습니다.

### 기본 통계 : 구역 방문 페이지

특정 구역에 방문한 사람 수와 그 체류 시간 등. 방문객과 매장 내 특정 구역에 대한 핵심 KPI를 제공합니다. 고객사는 특정 구역을 방문한 사람 수 대비 체류객 발생 비율 등의 데이터를 확인하고, 매대와 상품에 대한 매력도를 측정해 매장 운영 전략을 재고할 수 있습니다.

### 방문객 행동 : 방문객 동선 페이지

매장 방문객의 주요 동선을 한눈에 파악할 수 있습니다. 고객사는 방문객들의 주요 경로를 파악하여 공간 동선 기획 및 상품의 배치 방식을 검토하거나 매장 내 입지들의 가치를 측정할 수 있습니다.

위에서 소개한 4종의 페이지는 매쉬보드 좌측 메뉴 탭에서 즉시 사용할 수 있도록 제공됩니다. 상기 4종의 페이지를 활용하면 대시보드를 일일이 생성하거나 직접 편집할 필요 없이 매장 운영의 필수 지표를 손쉽게 확인할 수 있습니다. 따라서 **지표를 빠르게 조회하고 싶거나 데이터 분석이 처음인 사용자 혹은 매쉬보드 입문자에게 적극적인 이용을 권장합니다.**

## 사용자를 구속하지 않는 커스텀 기능

매쉬보드에는 사용자의 편의를 중점으로 구현된 기능 외에도 API를 통한 자유로운 데이터 조회, 데이터 조회의 목적과 용도에 따른 대시보드 커스텀 등의 심화 기능 또한 제공합니다. 그중에서도 메이아이가 특별히 공을 들이고 있는 부분은 바로 '커스텀 기능'입니다.

**커스텀 기능은 이름 그대로 사용자가 직접 데이터 조회 방식, 조건, 뷰어 스타일 등의 요소를 조작할 수 있도록 만들어진 기능을 의미합니다.** 앞서 소개한 4종의 페이지가 이미 생성되어 있는 대시보드를 이용하여 데이터를 간편하게 확인하는 것이 장점이라면, 커스텀 기능은 사용자가 원하는 정보를 선택적, 세부적, 집중적으로 볼 수 있다는 강점을 갖고 있습니다.

메이아이가 커스텀 기능에 공을 들이는 이유는 메이아이가 프로덕트를 대할 때 가장 중요하게 생각하는 가치가 바로 '데이터의 자유도'이기 때문입니다. 메이아이의 고객사는 각기 다른 업종, 업태, 주요 타깃, 매장의 입지 조건과 비즈니스 모델을 가지고 있습니다. 연쇄적으로 고객사마다 매장 운영 전략이나 확인하고 싶은 지표, 데이터의 사용 방식도 달라질 수밖에 없는데요. 이처럼 고객사의 니즈 편차가 큰 상황에서, 메이아이는 이를 모두 만족시키는 방법이 '매쉬보드 내에서의 데이터 자유도를 높이는 것'이라고 믿고 있습니다.

매쉬보드의 커스텀 기능은 크게 세 가지 방식으로 이용할 수 있습니다.

**첫 번째는 '대시보드' 페이지에서 제공하는 템플릿을 활용해 기본적인 차트가 포함된 대시보드를 생성한 뒤, 필요에 따라 위젯을 추가하는 방법입니다.** 매쉬보드가 제공하는 템플릿은 메이아이의 데이터 분석가가 미리 제작해 놓은 것으로, 사용자들의 주요 데이터 분석 목적에 따라 생성되어 있으며 전문가의 데이터 분석 노하우가 십분 반영되어 있습니다. 따라서 템플릿으로 간편하게 대시보드를 생성한 뒤 개개인의 취향에 맞게 위젯을 추가하면 전문가의 노하우를 기반으로 한 맞춤형 대시보드를 손쉽게 제작할 수 있습니다.

**두 번째는 '대시보드' 페이지에서 빈 대시보드를 생성한 후 위젯과 그래프를 하나씩 추가하는 방법입니다.** 기본 차트부터 위젯의 종류와 수, 위치 등. 대시보드를 구성하는 모든 요소를 사용자가 손수 편집하는 만큼 독자적인 레이아웃의 대시보드가 필요한 경우에 유용합니다. 다만 이 방법은 대시보드를 구성하는데 들여야 하는 공수가 큰 편이기 때문에 데이터 분석 경험 및 매쉬보드 사용 경험이 풍부한 사용자에게 활용을 권장합니다.

마지막 방법은 '커스텀 이벤트' 기능을 이용하는 것입니다. 커스텀 이벤트란 고객사의 요청에 따라 제공되는 부가 서비스로, **매쉬보드가 제공하는 데이터 지표 및 위젯 외에 별도로 원하는 항목이 생길 경우 메이아이에게 추가 개발을 요청하는 방식입니다.** 커스텀 이벤트 기능을 통해 전달된 사용자의 요청 사항은 긍정적인 내부 검토와 고객사와의 세부 협의를 거쳐 매쉬보드에 구현될 수 있습니다.

## 매쉬보드의 차별점을 빠르게 체감하고 싶다면

엔지니어, 디자이너, 데이터 분석가 등. 매쉬보드는 메이아이 내 분야별 전문가의 고민과 노력, 실험의 산물입니다. 매쉬보드가 편의성, 전문성 그리고 데이터의 자유도에서 기성의 데이터 분석 툴과 얼마나 격을 달리하는지 궁금하시다면 **매쉬보드 데모를 신청해 주시기를 제안 드립니다.** 매쉬보드 데모는 누구나 무료로 신청하실 수 있습니다. 데모를 신청해 주시면 담당자가 확인한 후 빠르게 도움을 드리도록 하겠습니다.
