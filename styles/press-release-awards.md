# 수상/논문 채택 보도자료 스타일 가이드

> 이 가이드는 메이아이의 수상 및 논문 채택 보도 기사 19편을 분석하여 작성됨.
> - CES 혁신상 수상 관련 기사 6편 (전자신문, 아시아경제, 매일신문, 서울경제TV, 데일리시큐, 시선뉴스)
> - 이머징 AI+X 톱 100 선정 관련 기사 4편 (유니콘팩토리, ZDNet, K스타트업밸리, 뉴스탭)
> - Good AI Awards 수상 관련 기사 2편 (더에이아이, 디지틀조선일보)
> - ICLR 2025 논문 채택 관련 기사 7편 (벤처스퀘어, 이데일리, ZDNet, 테크M, 디스커버리뉴스, K스타트업밸리, 베타뉴스)

---

## 1. 문체

### 어미 패턴
- **기본 어미:** `~했다`, `~밝혔다`, `~꼽힌다`, `~됐다`
- **수상/채택 서술:** `~을 수상했다`, `~에 채택됐다`, `~에 선정됐다`
- **가치 평가:** `~로 꼽힌다`, `~로 불린다`, `~을 인정받았다`
- **인용 연결:** `~라고 밝혔다`, `~라고 강조했다`, `~라고 전했다`
- **진행 중:** `~하고 있다`, `~활용 중이다`, `~제공 중이다`

### 격식 수준
- 투자 보도자료와 동일하게 **서술형 평서문** (`~했다` 체)
- 다만 수상/학회 권위를 설명할 때 약간의 **격식 높은 어조** 사용:
  ```
  'CES 최고의 영예'로 꼽힌다.
  글로벌 3대 AI 학회로 꼽힌다.
  ```
- 기술 성과를 서술할 때 `입증했다`, `인정받았다` 등 **성취 동사** 빈번

### 실제 예시
```
AI 스타트업 메이아이(mAy-I)가 CES 2024에서 혁신상(Innovation Award)을 수상했다.
ICLR은 뉴립스, 국제머신러닝학회와 함께 글로벌 3대 AI 학회로 꼽힌다.
메이아이가 제출한 논문은 상위 5%에 부여되는 '스포트라이트'로 선정돼 혁신성을 인정받았다.
```

## 2. 글 구조

### 수상 보도자료 (CES 혁신상, Good AI Awards, 이머징 AI+X 톱 100)

**전형적 5~6단 구성:**
1. **리드:** 수상/선정 핵심 사실 (누가 + 어디서 + 무엇을)
2. **상/행사 설명:** 해당 상의 권위와 행사 개요
3. **수상 제품/솔루션 설명:** 매쉬의 기능 상세 설명
4. **기술 성과 부가 정보:** 논문 채택, 특허 등 추가 성과
5. **향후 계획:** 시장 확대 방향
6. **대표 코멘트:** 포부/비전

### 논문 채택 보도자료 (ICLR)

**전형적 5~6단 구성:**
1. **리드:** 논문 채택 핵심 사실 (누가 + 어떤 학회 + 논문명/주제)
2. **학회 권위 설명:** ICLR이 3대 AI 학회인 점 + 스포트라이트 선정
3. **연구 내용 상세:** 논문이 다루는 기술적 문제와 해결책
4. **성능 데이터:** 자체 기술의 정확도 수치 (92% 등)
5. **관계자 코멘트 (2인):** 제1저자 + CPO
6. **부가 성과:** 미국 특허 취득 등

### 구조 차이점 (투자 보도자료 대비)
- 수상/학회의 **권위/배경 설명**에 상당한 분량 할애 (전체의 20~30%)
- 투자 보도자료의 "투자 참여사" 자리에 **"상/학회 설명"**이 위치
- 기술적 내용이 더 상세 (논문 보도자료의 경우)
- 인용문이 **2인 이상** 등장 가능 (논문 보도자료: 제1저자 + CPO)

## 3. 리드(첫 문단) 작성법

### 수상 보도자료 리드 공식
```
[수식어] [회사명]이/가 '[행사명]'에서 [상명]을 수상했다.
```

### 논문 채택 보도자료 리드 공식
```
[수식어] [회사명]의 [논문 주제] 논문이 [학회명]에 채택됐다[고 밝혔다].
```

### 선정 보도자료 리드 공식
```
[솔루션 설명] 개발사 [회사명]이 [주관기관] 주관 '[선정 제도명]'에 [N년 연속] 선정됐다.
```

### 실제 리드 예시

**CES 혁신상:**
```
AI 스타트업 메이아이(mAy-I)가 CES 2024에서 혁신상(Innovation Award)을 수상했다.
```
```
AI 스타트업 mAy-I(메이아이)가 'CES 2024'에서 혁신상을 수상했다고 발표했다.
```

**ICLR 논문:**
```
영상처리 인공지능 스타트업 (주)메이아이(대표 박준혁)의 '사람 재식별 AI 모델의 카메라 편향에 대한 연구' 논문이 세계 최고 권위 AI 학술대회인 '표현학습국제학회(ICLR) 2025'에 채택됐다.
```
```
메이아이는 '사람 재식별 AI 모델의 카메라 편향에 대한 연구' 논문이 AI 학술대회인 표현학습국제학회(ICLR) 2025에 채택됐다고 밝혔다.
```

**이머징 AI+X:**
```
영상처리 AI 솔루션 '매쉬' 개발사 메이아이가 한국인공지능산업협회(AIIA) 주관 '이머징 AI+X 톱 100'에 3년 연속 선정됐다.
```

### 리드 특징
- 수상/채택 보도자료는 리드에 **상의 공식 명칭**을 정확히 기재
- 영문명을 괄호로 병기: `혁신상(Innovation Award)`, `표현학습국제학회(ICLR)`
- 논문 보도자료는 리드에 **논문 주제**를 포함하는 경우가 많음
- `~을 수상했다`, `~에 채택됐다`, `~에 선정됐다` 등 성과 동사로 마무리

## 4. 자주 쓰는 표현/관용구

| 표현 | 사용 맥락 |
|------|-----------|
| `'CES 최고의 영예'로 꼽힌다` | CES 혁신상 권위 설명 |
| `미국소비자기술협회(CTA)가 수여하는 상` | CES 혁신상 주체 설명 |
| `세계를 선도할 기술력과 혁신성이 뛰어난 제품에 수여` | CES 혁신상 기준 설명 |
| `글로벌 3대 AI 학회로 꼽힌다` | ICLR 권위 설명 |
| `뉴립스(NeurIPS), 국제머신러닝학회(ICML)와 함께` | ICLR 동급 학회 나열 |
| `엄격한 심사를 거쳐 우수 논문을 선정한다` | 학회 심사 과정 설명 |
| `상위 5%에 부여되는 '스포트라이트(Spotlight)'` | 논문 등급 설명 |
| `혁신성을 인정받았다` | 성과 강조 마무리 |

### 기술/성과 표현
| 표현 | 사용 맥락 |
|------|-----------|
| `기술 경쟁력을 입증했다` | 리드 또는 마무리에서 성과 요약 |
| `기술력을 대외적으로 인정받았다` | 부가 성과 언급 시 |
| `세계 최고 수준의 기술력` | CPO/대표 코멘트 |
| `최신 학계 모델(ISR, 66%)을 능가하는 92%의 정확도` | 성능 비교 데이터 |
| `개인정보를 보호하면서 정밀한 매장 방문객 데이터를 제공하는 핵심 기술` | CPO 코멘트 정형 표현 |

### 사업 소개 관련 (투자 보도자료와 공통)
| 표현 | 사용 맥락 |
|------|-----------|
| `영상처리 AI 기반 오프라인 방문객 데이터 분석 솔루션` | 매쉬 한 줄 설명 |
| `오프라인 공간의 문제를 데이터 기반으로 해결` | 매쉬 가치 요약 |

| `방문객의 성별·연령대·동선·체류 시간·행동·상품 인터랙션 등을 분석` | 매쉬 분석 항목 나열 |
| `'mAsh Board(매쉬 보드)'라는 웹 대시보드로 제공` | 결과물 전달 방식 설명 |

## 5. 인용문 활용

### CES/수상 보도자료: 1인 인용
- **대표 코멘트** 1개가 보도자료 마지막에 배치
- 패턴:
  ```
  박준혁 대표는 "지속적으로 갈고 닦아 온 기술력을 바탕으로 글로벌 시장 진출에도 기민하게 접근할 것"이라고 밝혔다.
  ```

### Good AI Awards: 2인 인용
- **심사위원 코멘트** (중간) + **대표 코멘트** (마지막)
- 패턴:
  ```
  김봉제 심사위원: "메이아이의 AI 기술과 서비스는 독창적이면서도 실용적으로 평가됐다"
  박준혁 대표: "AI 기술과 안전성 모두 강화하며 성장해 나가겠다"
  ```

### 논문 채택 보도자료: 2인 인용
- **제1저자(연구팀 리드) 코멘트** + **CPO 코멘트**
- 배치: 연구 내용/성과 설명 직후, 보도자료 후반부에 연속 배치
- 패턴:
  ```
  제1저자 송명서 연구팀 리드는 "매쉬의 정확도 향상을 위한 연구가 세계적으로 인정받아 기쁘다"며 더 많은 고객에게 신뢰할 수 있는 분석 결과를 제공할 수 있을 것으로 기대한다고 밝혔다.
  
  김찬규 CPO는 해당 기술이 "개인정보를 보호하면서 정밀한 매장 방문객 데이터를 제공하는 메이아이의 핵심 기술"이라고 강조했다.
  ```

### 이머징 톱100 보도자료: 1인 인용
- **대표 코멘트**만 보도자료 마지막에 배치
- 패턴:
  ```
  박준혁 대표는 "3년 연속 선정으로 영상처리 AI 분석 업계 선두주자로 인정받게 됐다"며, 국내외 오프라인 매장을 더욱 스마트하게 만드는 데 주력하겠다고 밝혔다.
  ```

### 인용 형식 규칙
- 핵심 키워드/문구만 큰따옴표로 직접 인용
- 나머지는 간접 인용으로 연결
- 인용 마무리 동사: `~라고 밝혔다`, `~라고 강조했다`, `~라고 전했다`, `~라고 말했다`

## 6. 숫자/데이터 표기법

### 날짜/시기
- 행사 일정: `2024년 1월 9일~12일`, `2024년 1월 9~12일`
- 본문 내: `6일 밝혔다`
- 연도: `2023년`, `2024~2025년`

### 수량/규모
- 참가 규모: `3,500개 이상의 참관사`, `500개 이상 기업`
- 횟수: `3년 연속`, `올해로 3회째`
- 순위: `상위 5%`

### 정확도/성능
- 퍼센트: `92%의 정확도`, `66%` — 아라비아 숫자 + % 기호
- 비교 표현: `최신 학계 모델(ISR, 66%)을 능가하는 92%의 정확도`
- 모델명과 수치를 괄호로 병기: `(ISR, 66%)`

### 고유명사 표기
- 영문명 괄호 병기: `표현학습국제학회(ICLR)`, `미국소비자기술협회(CTA)`, `뉴립스(NeurIPS)`
- 한영 혼용: `CVPR(Computer Vision and Pattern Recognition) 2023`
- 상명 병기: `혁신상(Innovation Award)`, `스포트라이트(Spotlight)`

## 7. 회사 소개 패턴

### CES/수상 보도자료에서의 첫 등장
```
AI 스타트업 메이아이(mAy-I)
AI 스타트업 mAy-I(메이아이)
AI(인공지능) 스타트업 mAy-I(메이아이)
```
- 투자 보도자료보다 수식어가 짧음: `AI 스타트업` (vs 투자 보도자료의 `영상 처리 AI 스타트업`)

### 논문 보도자료에서의 첫 등장
```
영상처리 인공지능 스타트업 (주)메이아이(대표 박준혁)
영상처리 AI 스타트업 메이아이
메이아이(대표 박준혁)
```

### 톱100 선정 보도자료에서의 첫 등장
```
영상처리 AI 솔루션 '매쉬' 개발사 메이아이
영상처리 AI 솔루션 '매쉬(mAsh)' 개발사 메이아이
```
- 회사보다 **솔루션을 먼저** 언급하는 패턴 (솔루션이 선정 대상이므로)

### 솔루션(매쉬) 설명 — 수상 보도자료에서 더 상세
```
mAsh(매쉬)는 영상처리 AI 기반 오프라인 방문객 데이터 분석 솔루션으로, 오프라인 공간의 문제를 데이터 기반으로 해결하고 방문객 데이터 기반 의사결정을 지원한다.
```
- 수상 보도자료는 투자 보도자료보다 매쉬의 **기능을 더 풍부하게** 설명
- `mAsh Board(매쉬 보드)`, `커스텀 대시보드`, `구역 분석`, `고객 여정 분석`, `인사이트 리포트` 등 구체적 기능 언급
- 논문 보도자료는 매쉬보다 **해당 논문의 기술**에 초점

### 고객사 언급
- CES 보도자료: `국내 유수의 고객사` (구체 사명 없이)
- 톱100 보도자료: `삼성전자, LG전자, CGV 등`

## 8. 문장 길이와 단락 구성

### 문장 길이
- 평균 **45~80자** (투자 보도자료보다 약간 김)
- 리드 문장: 약 30~70자
- 상/학회 설명 문장: 약 50~90자 (고유명사, 부연 설명으로 길어짐)
- 기술 설명 문장: 약 60~100자 (논문 보도자료에서 특히 김)
- 인용 문장: 약 50~80자

### 단락 구성
- **단락당 1~3문장**
- 상/학회 배경 설명은 2~4문장으로 하나의 블록 형성
- 기술 설명도 2~3문장으로 하나의 블록
- 인용문은 각각 별도 단락

### CES 보도자료 특유의 배경 설명 블록
```
CES는 세계 최대 IT·가전 박람회로, 2024년 1월 9~12일 미국 라스베이거스에서 열리며 주제는 '올 투게더, 올 온'이다. 3,500개 이상의 참관사가 참여하며, 국내에서는 삼성·LG전자·현대차 등 500개 이상 기업이 참가한다.
```
- 이 블록이 거의 모든 CES 보도자료에서 동일한 내용으로 반복됨

### ICLR 보도자료 특유의 기술 설명 블록
```
해당 논문에서는 CCTV별 환경 차이로 인해 동일 인물 판별이 어려운 기존 AI 모델의 한계를 분석하고, 다양한 조건에서도 일관된 정확도를 유지할 수 있도록 카메라 간 편차 완화 방법을 연구했다.
```
- 기술 내용을 비전문가도 이해할 수 있는 수준으로 풀어 서술

## 12. 글 생성 시 체크리스트

수상/논문 채택 보도자료를 생성할 때 아래 항목을 유형별로 확인:

### 공통
- [ ] 리드에 회사명 + 수상/채택/선정 사실 명확히 기재
- [ ] 상/학회의 영문명 괄호 병기
- [ ] 상/학회/제도의 권위 설명 포함
- [ ] 솔루션(매쉬) 소개 포함
- [ ] 대표(또는 관계자) 코멘트로 마무리
- [ ] `~했다` 서술체 사용

### CES/수상 보도자료 추가
- [ ] 행사 일정, 장소, 주제, 참가 규모 등 배경 정보
- [ ] 혁신상의 정의/의미 설명
- [ ] 매쉬의 기능 상세 설명 (대시보드, 분석 항목 등)
- [ ] 향후 시장 확대 계획
- [ ] 전체 450~700자, 5~8문단

### ICLR/논문 보도자료 추가
- [ ] 학회가 3대 AI 학회 중 하나임을 명시 (NeurIPS, ICML과 함께)
- [ ] 논문 등급 (스포트라이트 = 상위 5%) 명시
- [ ] 논문의 기술적 내용을 비전문가 수준으로 풀어 설명
- [ ] 자체 기술 성능 수치 (92% 정확도 등) 포함
- [ ] 제1저자 코멘트 + CPO 코멘트 (2인)
- [ ] 부가 성과 (미국 특허 등) 언급
- [ ] 전체 400~750자, 5~8문단

### 이머징 톱100 보도자료 추가
- [ ] 주관기관(AIIA) 명시
- [ ] 선정 제도의 평가 기준 설명
- [ ] 연도별 선정 분야 이력
- [ ] 주요 고객사 나열
- [ ] 글로벌 현황 (해외 거점, 특허 등)
- [ ] 전체 350~550자, 4~7문단
