# 스타일 가이드: 정부 지원 사업 선정 보도자료 (Government Program Press Release)

> 메이아이의 정부 지원 프로그램(포스트팁스, 스케일업팁스 등) 선정 소식을 다루는 보도자료의 작성 스타일 가이드.
> 분석 대상: 디지털데일리, 전자신문, 이데일리, 디스커버리뉴스, 유니콘팩토리, K스타트업밸리, 디지털투데이, 벤처스퀘어, 한스경제, 이뉴스투데이 등 보도 기사 10편

---

## 1. 문체

### 어미 패턴
- **기본 어미:** `~다` 체 (서술체)
- 주로 사용되는 어미:
  - `~선정됐다`, `~선정됐다고 밝혔다` (핵심 뉴스 전달)
  - `~인정받아 선정됐다` (선정 사유)
  - `~지원받는다`, `~지원받게 된다` (지원금 안내)
  - `~도입돼 있다`, `~공급 중이다` (현재 상태)
  - `~추진할 계획이다`, `~추진할 방침이다` (향후 계획)
  - `~파악하는 솔루션이다` (제품 정의)

### 격식 수준
- 실적 보도자료와 동일한 **객관적 서술체**
- 정부 프로그램 설명 부분에서 약간 더 공식적/설명적 톤
- 감탄이나 주관적 평가 배제

### 실제 예시
```
메이아이가 중소벤처기업부 주관 스케일업 지원 프로그램 포스트팁스(Post-TIPS)에 최종 선정됐다.
```
```
선정 기업에는 향후 18개월간 최대 7억 원의 정부 지원금이 제공된다.
```
```
7월 포스트 팁스 선정에 이어 2개월 만의 쾌거다.
```

## 2. 글 구조

### 전형적 구조 (5~6단락)

```
[리드] 회사명 + 프로그램 선정 사실
[본문1] 프로그램 설명 (무엇인지, 지원 규모)
[본문2] 선정 사유 (기술 혁신성, 글로벌 확장성)
[본문3] 매쉬 솔루션 소개 + 주요 고객사
[본문4] 기술적 차별점 (엣지 컴퓨팅, GDPR 준수, 해외 공급)
[본문5] 향후 계획
[마무리] 대표 인용문
```

### 실적 보도자료와의 차이
- **프로그램 설명 단락이 추가됨:** 포스트팁스/스케일업팁스가 무엇인지 1단락으로 설명
- **선정 사유를 명시함:** `~의 기술 혁신성과 글로벌 확장성을 인정받아 선정됐다`
- **지원금 규모가 반드시 포함됨:** `18개월간 최대 7억 원` / `3년간 총 12억원`
- 솔루션 소개가 리드 바로 뒤가 아닌, 프로그램 설명 뒤에 위치

## 3. 리드(첫 문단) 작성법

### 패턴
**[회사 수식어(선택)] + [회사명(대표명)] + [이/가] + [정부 부처명] + [주관] + [프로그램명] + [에 최종 선정됐다(고 밝혔다)]**

### 정보 배치 순서
1. 회사 수식어 (선택적 - 있을 때: "영상처리 AI 스타트업", "매장 분석 AI 솔루션 '매쉬'를 운영하는")
2. 회사명 + 대표명 (예: "메이아이(대표 김찬규)", "㈜메이아이(대표 김찬규)")
3. 정부 부처명 (예: "중소벤처기업부 주관")
4. 프로그램명 (예: "스케일업 지원 프로그램 '포스트팁스(Post-TIPS)'")
5. 선정 동사 + 날짜 (예: "에 최종 선정됐다고 28일 밝혔다")

### 리드 실제 예시
```
메이아이가 중소벤처기업부 주관 스케일업 지원 프로그램인 '포스트팁스(Post-TIPS)'에 최종 선정되었다고 28일 발표했다.
```
```
매장 분석 AI 솔루션 '매쉬'를 운영하는 메이아이가 중소벤처기업부 주관 스케일업 지원 프로그램 포스트팁스(Post-TIPS)에 최종 선정됐다.
```
```
영상처리 AI 스타트업 메이아이가 중소벤처기업부 주관 '스케일업 팁스'에 최종 선정됐다고 13일 밝혔다.
```
```
AI 기반 영상 분석 스타트업 ㈜메이아이(대표 김찬규)가 중소벤처기업부 주관 '스케일업 팁스(Scale-up TIPS)'에 최종 선정됐다고 13일 밝혔다.
```

### 리드 변형: 제품 기반 소개형
- 회사 수식어 대신 제품으로 회사를 소개하는 패턴이 자주 등장
```
매장 분석 AI 솔루션 '매쉬'를 운영하는 메이아이가 ~
```

## 4. 자주 쓰는 표현/관용구

### 선정 관련
- `최종 선정됐다` / `최종 선정됐다고 밝혔다`
- `~의 기술 혁신성과 글로벌 확장성을 인정받아 선정됐다`
- `~을 인정받아 선정됐으며`
- `높은 평가를 받았다`

### 프로그램 설명
- `팁스(TIPS) 졸업 기업 중 우수한 R&D 성과를 보유한 스타트업을 선별해`
- `규모 확장을 돕는 프로그램이다`
- `스케일업을 지원하는 프로그램이다`
- `유니콘 수준으로 육성하는 프로그램이다`
- `정부-민간 협력 프로그램`
- `중기부 대표 민관합동 창업 육성 프로그램`

### 지원금 표현
- `향후 18개월간 최대 7억 원의 정부 지원금을 받게 된다`
- `향후 3년간 총 12억원 규모의 R&D 자금을 지원받는다`
- `선정 기업에는 ~의 정부 지원금이 제공된다`

### 기술/제품 설명
- `딥러닝 기반 알고리즘을 활용해`
- `매장 내 CCTV를 분석해`
- `방문객 반응·동선·체류 시간, 직원·방문객 간 상호작용 등을 파악`
- `기존 수기 조사나 WiFi 센서 기반 분석의 한계를 극복`

### 글로벌/규제 관련
- `EU GDPR 및 AI Act 규제를 준수`
- `엣지 컴퓨팅 기술`
- `미국·영국·베트남 등 해외 매장에 공급 중`
- `국제 개인정보보호 기준을 충족`

### 계획/전망 표현
- `~고도화를 추진할 계획이다`
- `글로벌 브랜드와의 협업 강화`
- `본격적인 글로벌 진출을 이어갈 방침이다`
- `시장 점유율 확대`

### 연속 성과 강조
- `7월 포스트 팁스 선정에 이어 약 2개월 만의 추가 성과다`
- `2개월 만의 쾌거`
- `연이어 주요 정부 지원 프로그램에 선정된 것은`

## 5. 인용문 활용

### 배치 위치
- **항상 보도자료 맨 마지막 단락**에 위치 (실적 보도자료와 동일)
- 보도자료 중간에 인용문 등장 없음

### 인용문 형식
**[이름] + [직책(대표)] + [는/은] + [이번 선정이] + ["직접 인용"] + [발화 동사]**

### 발화 동사
- `~라고 밝혔다` / `~라며 ~밝혔다` (가장 빈번)
- `~라는 포부를 밝혔다`
- `~라는 의지를 밝혔다`
- `~라고 전했다`

### 인용문 실제 예시
```
김찬규 대표는 이번 선정이 "AI 기술력과 글로벌 시장 진출 역량을 공식적으로 인정받은 결과"라며 리테일테크 혁신을 주도하는 기업으로 성장하겠다는 포부를 밝혔다.
```
```
김찬규 대표는 "이번 포스트팁스 선정은 메이아이의 AI 기술력과 글로벌 시장 진출 역량을 공식적으로 인정받은 결과"라며 글로벌 리테일테크 혁신 기업으로의 성장 의지를 밝혔다.
```
```
김찬규 대표는 "2개월 사이 연이어 주요 정부 지원 프로그램에 선정된 것은 기술력과 시장성을 동시에 인정받은 결과"라며, 리테일 기술 시장에서 실질적 성과를 창출하는 기업으로 성장하겠다는 포부를 밝혔다.
```
```
김찬규 대표는 "단단한 기술 기반과 국내 시장에서 수년간 증명된 매쉬의 효용성 덕분"이라며 "리테일 테크 선도 기업으로서 성과 창출에 최선을 다하겠다"고 밝혔다.
```

### 인용문 내용 패턴
- **포스트팁스 보도자료:** `이번 선정` + `AI 기술력과 글로벌 시장 진출 역량을 공식적으로 인정받은 결과` + `리테일테크 혁신 주도`
- **스케일업팁스 보도자료:** `연이은 선정` + `기술력과 시장성 동시 인정` + `실질적 성과 창출`
- 실적 보도자료의 인용문보다 **"공식 인정"**을 강조하는 톤

## 6. 숫자/데이터 표기법

### 지원금
- `최대 7억원` / `최대 7억 원` (원 앞 띄어쓰기 비통일)
- `총 12억원 규모` / `총 12억 원 규모`

### 기간
- `18개월간` / `향후 18개월간`
- `3년간` / `향후 3년간`

### 날짜
- `28일 발표했다` / `28일 밝혔다` (일만 표기)
- `2025년 7월 28일` (메타데이터)
- `지난 9월` / `2024년부터`

### 퍼센트/비율
- 이 유형에서는 상대적으로 수치 데이터가 적음
- 수치보다 **정성적 성과**(선정, 채택, 인정)가 중심

### 순위/등급
- `상위 5%` / `상위 5%인 '스포트라이트(Spotlight)'`

## 7. 회사 소개 패턴

### 첫 등장 (리드) - 두 가지 패턴

#### 패턴 A: 수식어 + 회사명
```
영상처리 AI 스타트업 메이아이가 ~
```
```
AI 기반 영상 분석 스타트업 ㈜메이아이(대표 김찬규)가 ~
```

#### 패턴 B: 제품으로 회사 소개 (이 유형에서 더 빈번)
```
매장 분석 AI 솔루션 '매쉬'를 운영하는 메이아이가 ~
```
```
매장 분석 AI 솔루션 '매쉬'를 운영하는 ㈜메이아이(대표 김찬규)가 ~
```

### 특징
- 실적 보도자료에서는 패턴 A가 주류이나, 정부 지원 보도자료에서는 **패턴 B(제품 기반 소개)**가 절반 이상
- 이는 독자에게 "무엇을 하는 회사인지"를 리드에서 바로 전달하려는 의도

### 매쉬 소개 전형적 문장
```
매쉬는 딥러닝 기반 알고리즘을 활용해 매장 내 CCTV를 분석, 특정 구역의 방문객 반응·동선·체류 시간, 직원과 방문객 간 상호작용 등을 정밀 파악하는 솔루션이다.
```
```
매쉬는 CCTV 영상 기반 딥러닝 분석 기술로 방문객 동선, 체류 시간, 구역별 반응, 직원·고객 간 상호작용 등을 추적·분석하는 솔루션이다.
```

### 고객사 언급
- **포스트팁스 보도자료:** `LG유플러스, CGV 등 대기업에 도입`
- **스케일업팁스 보도자료:** `삼성전자, LG전자, 스파오, 코웨이 등에 공급 중`
- 고객사 목록이 시기에 따라 업데이트됨

## 8. 문장 길이와 단락 구성

### 문장 길이
- 평균 **40~80자** 내외
- 프로그램 설명 문장이 상대적으로 길어지는 경향 (~80자)
- 짧은 문장: `현재 LG유플러스, CGV 등에 솔루션을 공급 중이다.` (~25자)
- 긴 문장: `매쉬는 딥러닝 기반 알고리즘을 활용해 매장 내 CCTV를 분석, 특정 구역의 방문객 반응·동선·체류 시간, 직원과 방문객 간 상호작용 등을 정밀 파악하는 솔루션이다.` (~90자)

### 단락당 문장 수
- **1~3문장**이 일반적
- 프로그램 설명 단락: 2~3문장
- 솔루션 소개 단락: 1~2문장
- 인용문 단락: 1문장

### 전반적 톤
- 실적 보도자료보다 **설명적 문장 비중이 높음** (프로그램 설명, 기술 설명)
- 정량 데이터보다 **정성적 성과/인정 사실** 중심

---

## 9. 전체 글 길이

- **짧은 보도자료:** 약 250~350자 (전자신문, 디지털투데이, 유니콘팩토리 스타일)
- **중간 보도자료:** 약 400~550자 (디지털데일리, K스타트업밸리, 이뉴스투데이 스타일)
- **긴 보도자료:** 약 600~800자 (이데일리, 디스커버리뉴스, 벤처스퀘어, 한스경제 스타일)
- **단락 수:** 4~7개 단락
- 실적 보도자료보다 전반적으로 **10~20% 짧은 경향**


### 제목 특징
- `중기부`로 약칭 사용 (중소벤처기업부 -> 중기부)
- 프로그램명이 반드시 포함 (포스트팁스, 스케일업팁스)
- `선정`이 핵심 동사
- `최종 선정`으로 확정 뉘앙스 강조
- `…` 뒤에 부연 설명(기술 고평가, AI 솔루션 등) 추가
- 실적 보도자료와 달리 **숫자(%)가 제목에 거의 등장하지 않음**

## 11. 실적 보도자료와의 핵심 차이 요약

| 항목 | 실적(Performance) 보도자료 | 정부 지원(Government) 보도자료 |
|------|----------------------|------------------------|
| 리드 핵심 | 매출/성장률 수치 | 프로그램 선정 사실 |
| 회사 소개 | 수식어 + 회사명 | 제품명으로 회사 소개 빈번 |
| 필수 포함 정보 | 매출 수치, 성장률 | 프로그램명, 지원금 규모, 지원 기간 |
| 프로그램 설명 | 없음 | 1단락 할애 (프로그램 정의 + 지원 규모) |
| 데이터 비중 | 높음 (%, 배수, 시간 등) | 낮음 (지원금 외 정량 데이터 적음) |
| 인용문 톤 | 성과 + 비전 | "공식 인정" 강조 + 비전 |
| 글 길이 | 상대적으로 김 | 상대적으로 짧음 |
| 제목 숫자 | %↑ 등 수치 포함 | 수치 거의 없음 |
