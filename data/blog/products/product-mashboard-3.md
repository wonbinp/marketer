# 2분 만에 업무 생산성을 올려주는 매쉬보드 기능 3종

*프로덕트 디자이너가 추천하는 매쉬보드 핵심 기능 3종!*

---

안녕하세요, 메이아이의 Product Designer 정두리입니다.

2024년 새해를 맞아 목표 달성에 도움이 되는 매쉬보드 핵심 기능 3가지를 소개합니다.

---

## 1. 조회도 조합도 간편하게 -- 대시보드 템플릿과 위젯

사용자가 자주 쓰는 위젯을 목적에 따라 모아 놓은 영역이 바로 **템플릿**입니다.

매쉬보드 사용자 대부분은 '모든 매장' 메뉴의 기본 정보만 확인합니다. 위젯을 직접 조합하는 것이 어렵게 느껴지기 때문인데요. 템플릿 기능을 이용하면 목적에 맞는 템플릿을 클릭하는 것만으로 대시보드가 자동 생성됩니다.

템플릿은 다음 두 가지 방법으로 접근할 수 있습니다:
- 메인 페이지 내 **'빠른 생성'** 영역의 카드 클릭
- **'새 대시보드 생성'** 버튼 클릭

템플릿마다 설명이 적혀 있어 선택이 어렵지 않습니다. 추가로 보고 싶은 데이터가 생기면 **'위젯 추가'** 기능으로 대시보드를 자유롭게 편집할 수 있습니다.

---

## 2. 복잡한 차트를 한 줄의 핵심으로 -- 노트 기능

**노트** 기능의 핵심은 변수를 입력하는 간단한 조작으로 차트의 최대·최소·평균·합계를 자동으로 계산해준다는 점입니다.

기존에는 일별 평균값을 구하려면 직접 수치를 더한 뒤 날짜 수로 나눠야 했지만, 노트 기능을 사용하면 변수 하나만 추가하면 자동 계산됩니다.

**사용 방법:**
- '노트' 위젯을 추가하거나
- '차트 설정' 위젯에서 '보조 차트 토글'을 활성화

노트에 슬래시(/)를 입력하거나 변수 목록에서 직접 선택하면 텍스트를 남길 수 있으며, 폰트 크기·색상·정렬 방식 편집 및 링크 연결도 가능합니다.

실제 활용 사례:
- 차트에서 얻은 인사이트 기록
- 팀원에게 메모 남기기
- 대시보드·위젯별 설명 추가
- 대시보드 제목 텍스트 작성

---

## 3. 성과 측정의 중요성을 아는 일잘알을 위해 -- KPI 기능

**KPI 기능**을 활용하면 목표별 달성률을 번거롭게 계산하지 않고 한눈에 확인할 수 있습니다. 매장 실적을 체계적으로 추적하고 팀 성과를 모니터링하는 데 유용합니다.

**사용 방법 두 가지:**

1. 위젯의 차트 유형을 **'KPI'** 로 설정 -- 달성률을 강조해서 볼 때 적합
2. 차트 유형을 **'값'** 으로 선택 후 KPI 옵션 활성화 -- 달성률보다 상세 수치를 강조할 때 적합

---

## 마치며 -- 아끼는 식물에 물을 주는 마음으로

데이터를 다루는 것은 쉽지 않습니다. 어떻게 하면 사용자가 매쉬보드 안에서 더 쉽게 데이터를 다루고 인사이트를 발견할 수 있을지 고민합니다.

매쉬보드를 잘 활용해 주시는 모습을 볼 때면, 좋아하는 식물에 물을 주듯 더 멋진 프로덕트로 키우고 싶다는 마음이 자라납니다. 앞으로도 더 유용한 프로덕트를 만들 수 있도록 노력하겠습니다. 성장 기록은 **'mAsh 업데이트 노트'** 에 모아두겠습니다.
