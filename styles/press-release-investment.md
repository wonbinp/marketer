# 투자 유치 보도자료 스타일 가이드

> 이 가이드는 메이아이의 시리즈A 투자 유치 보도 기사 12편(벤처스퀘어, 인공지능신문, AI타임스, K글로벌타임스, 스타트업투데이, 디지털투데이, 스타트업데일리, 유니콘팩토리, 한국M&A경제신문, 정보통신신문, 시사의창, The SaaS News)을 분석하여 작성됨.

---

## 1. 문체

### 어미 패턴
- **기본 어미:** `~했다`, `~밝혔다`, `~전했다` (과거형 서술체)
- **현재 상태 서술:** `~하고 있다`, `~제공한다`, `~활용한다`
- **계획/예정:** `~할 계획이다`, `~할 예정이다`, `~에 활용할 계획이다`
- **인용 연결:** `~라고 평가했다`, `~라고 밝혔다`, `~는 포부를 밝혔다`, `~는 의지를 밝혔다`

### 격식 수준
- 경어체가 아닌 **서술형 평서문** (합쇼체/해요체 사용하지 않음)
- 감정적 표현 최소화, 사실 중심의 객관적 톤
- `~라고 전했다`, `~라고 밝혔다` 등 간접 인용이 기본

### 실제 예시
```
영상 처리 AI 스타트업 메이아이(mAy-I)가 60억 원 규모의 시리즈A 투자 유치를 완료했다.
메이아이는 투자금을 고객 확장, AI 기술 고도화, 글로벌 진출에 활용할 계획이다.
박준혁 대표는 글로벌 오프라인 방문객 데이터 분석 리더로 자리매김하겠다는 포부를 밝혔다.
```

## 2. 글 구조

### 전형적 구조 (5단 구성)
1. **리드 (1문단):** 투자 유치 핵심 사실 (누가 + 얼마 + 라운드 + 완료)
2. **투자 참여사 (2문단):** 리드 투자자, 신규/후속 투자자, 누적 투자액
3. **투자자 코멘트 (3문단):** 심사역의 투자 배경/평가 발언
4. **사업 소개 (4문단):** 핵심 솔루션(매쉬) 설명 + 고객사
5. **향후 계획 + 대표 코멘트 (5문단):** 투자금 활용 계획 + 대표 발언

### 구조 변형
- 일부 보도자료는 **투자자 코멘트**가 사업 소개 뒤에 오기도 함
- 간결한 보도자료는 3~4단 구성 (투자자 코멘트 생략)
- CES 수상 등 **부가 성과**가 있으면 향후 계획 직전에 배치

## 3. 리드(첫 문단) 작성법

### 필수 요소 (한 문장에 모두 포함)
1. 회사 수식어: `영상 처리 AI 스타트업`
2. 회사명: `메이아이(mAy-I)` 또는 `메이아이(mAy-I, 대표 박준혁)`
3. 투자 금액: `60억 원 규모`
4. 라운드: `시리즈A`
5. 완료 동사: `투자 유치를 완료했다` / `투자 유치를 완료했다고 밝혔다`

### 리드 공식
```
[수식어] [회사명]이/가 [금액] 규모의 [라운드] 투자 유치를 [완료했다/마쳤다고 밝혔다].
```

### 실제 리드 예시
```
영상 처리 AI 스타트업 mAy-I(메이아이)가 60억 원 규모의 시리즈A 투자 유치를 완료했다.
```
```
AI 영상 처리 스타트업 메이아이(mAy-I, 대표 박준혁)가 60억 원 규모의 시리즈A 투자 유치를 완료했다고 15일 발표했다.
```
```
영상 AI 전문기업 메이아이(대표 박준혁)가 60억원 규모의 시리즈A 투자 유치를 마쳤다고 밝혔다.
```

### 리드에서 날짜 처리
- `15일 발표했다`, `15일 밝혔다` 등 날짜를 동사 앞에 삽입하는 경우 있음
- 날짜를 생략하는 보도자료도 많음 (절반 정도)

## 4. 자주 쓰는 표현/관용구

### 투자 관련
| 표현 | 사용 맥락 |
|------|-----------|
| `투자 유치를 완료했다` | 리드 문장 마무리 |
| `투자를 주도했으며` | 리드 투자자 소개 |
| `신규 참여했다` / `신규 투자자로 참여했다` | 새 투자자 소개 |
| `후속 투자를 진행했다` / `후속 투자를 이어갔다` | 기존 투자자 |
| `누적 투자액이 80억 원을 넘어섰다` | 누적 금액 강조 |
| `투자 한파 속에 이뤄졌다는 점에서 더 의미가 있다` | 대표 코멘트 필수 표현 |

### 사업 소개 관련
| 표현 | 사용 맥락 |
|------|-----------|
| `CCTV 기반 방문객 데이터 분석 솔루션` | 매쉬 한 줄 설명 |
| `방문객의 성별·연령대·동선·체류 시간·행동·상품 인터랙션 등을 분석` | 매쉬 기능 설명 |
| `오프라인 공간의 문제를 데이터로 해결` | 매쉬 가치 요약 |
| `유의미한 데이터를 제공할 수 있는 최적의 업체` | 삼성벤처투자 심사역 인용 |
| `오프라인 매장이 체험형 공간으로 변모하는 흐름` | 투자 배경 설명 |

### 향후 계획 관련
| 표현 | 사용 맥락 |
|------|-----------|
| `고객 확장, 사업 개발, AI 기술 고도화` | 투자금 활용 3대 축 |
| `글로벌 진출에 활용할 계획이다` | 해외 진출 언급 |
| `글로벌 오프라인 방문객 데이터 분석 리더로 자리매김하겠다` | 대표 포부 마무리 |

## 5. 인용문 활용

### 인용문 종류와 배치
1. **투자자(심사역) 코멘트** — 본문 중반부 (투자 참여사 소개 직후)
2. **대표 코멘트** — 보도자료 마지막 문단 (클로징)

### 인용 형식
- **간접 인용**이 압도적 다수:
  ```
  삼성벤처투자 심사역은 메이아이를 "유의미한 데이터를 제공할 수 있는 최적의 업체"로 평가했다.
  ```
- 인용문은 큰따옴표(`""`)로 감싸며, 핵심 키워드만 직접 인용하고 나머지는 간접 서술로 연결:
  ```
  박준혁 대표는 이번 투자가 "투자 한파 속에 이뤄졌다는 점에서 더 의미가 있다"며, 글로벌 리더로 성장하겠다는 의지를 밝혔다.
  ```

### 인용 마무리 동사
- `~라고 평가했다`, `~라고 밝혔다`, `~는 포부를 밝혔다`, `~는 포부를 전했다`, `~는 의지를 밝혔다`, `~고 강조했다`

## 6. 숫자/데이터 표기법

### 금액
- **한글+숫자 혼용:** `60억 원`, `80억 원` (아라비아 숫자 + 한글 단위)
- 띄어쓰기: `60억 원` (숫자와 '원' 사이 공백)
- 일부 매체: `60억원` (공백 없음) — 두 형태 모두 사용됨
- 달러 표기 (해외 매체): `약 500만 달러`

### 퍼센트/비율
- `상위 5%` — 아라비아 숫자 + % 기호

### 날짜
- 본문 내: `15일`, `12월 15일` (월+일 또는 일만)
- 메타 정보: `2023년 12월 15일`

### 목록/나열
- 중간점(·) 사용: `에버그린투자파트너스·미래에셋벤처투자·플럭스벤처스`
- 쉼표 사용: `고객 확장, 사업 개발, AI 기술 고도화`
- 회사 나열 시 중간점, 일반 항목 나열 시 쉼표

## 7. 회사 소개 패턴

### 첫 등장 시 (리드)
```
영상 처리 AI 스타트업 메이아이(mAy-I)
영상 처리 AI 스타트업 메이아이(mAy-I, 대표 박준혁)
영상 AI 전문기업 메이아이(대표 박준혁)
영상처리 AI 스타트업 메이아이
```
- **패턴:** `[분야 수식어] [업종] [회사명(영문명)]`
- 수식어: `영상 처리 AI`, `영상처리 AI`, `AI 영상 처리`, `영상 처리 특화 AI`
- 업종: `스타트업`, `전문기업`
- 괄호 안: `mAy-I` 또는 `mAy-I, 대표 박준혁` 또는 `대표 박준혁`

### 솔루션 소개 시
```
CCTV 기반 방문객 데이터 분석 솔루션 'mAsh(매쉬)'
영상 처리 AI 솔루션 매쉬(mAsh)
매장 CCTV를 활용해 방문객 데이터를 분석하는 영상 처리 AI 솔루션
```

### 고객사 나열 패턴
```
현대자동차그룹·교보문고·신세계·이랜드리테일 등이 도입한 상태다.
현대자동차그룹, 교보문고, 신세계, 이랜드리테일 등이 주요 고객사다.
```
- 대형/유명 기업명 3~4개를 나열 후 `등` 으로 마무리

## 8. 문장 길이와 단락 구성

### 문장 길이
- 평균 **40~70자** (한글 기준)
- 리드 문장: 약 35~55자 (짧고 핵심적)
- 설명 문장: 약 60~90자 (솔루션 기능 나열 시 길어짐)
- 인용 문장: 약 50~80자

### 단락 구성
- **단락당 1~3문장**이 일반적
- 한 문단이 4문장을 넘는 경우는 드묾
- 투자 참여사는 한 문단에 2~3문장으로 리드/신규/후속을 순차 서술
- 솔루션 설명도 1~2문장으로 압축

### 단락 간 전환
- 명시적 접속사 최소화
- 주제 전환 시 새 단락으로 분리 (접속사 없이)
- 가끔 `이로써`, `또한`, `한편` 으로 연결

## 9. 전체 글 길이

- **본문 기준:** 약 300~600자 (한글 기준)
- **짧은 보도자료:** 약 250~350자 (벤처스퀘어, K글로벌타임스, 유니콘팩토리)
- **중간 보도자료:** 약 400~500자 (AI타임스, 디지털투데이, 한국M&A경제신문)
- **긴 보도자료:** 약 500~700자 (스타트업투데이, 스타트업데일리, 인공지능신문)
- **문단 수:** 4~8개
- **전체 문장 수:** 8~15문장

## 10. 제목 패턴

### 공식
```
[회사명], [금액] [라운드] 투자 유치 [부가 정보]
```

### 실제 제목 예시
```
메이아이, 60억 원 투자 유치 완료
AI 영상 스타트업 메이아이, 삼성벤처투자 리드 시리즈 A로 60억 원 투자유치
메이아이, 영상 처리 AI 기술로 60억 투자 유치
메이아이, 60억 원 규모 시리즈 A 투자 유치...오프라인 방문객 데이터 분석 리더로 자리매김
메이아이, 'CCTV AI 분석기술' 60억원 투자유치
```

### 제목 특징
- 회사명이 항상 먼저 등장
- 금액이 제목에 반드시 포함
- `...` 뒤에 포부/비전을 붙이는 패턴 빈번
- `유치`, `투자유치`, `투자 유치` 혼용

## 11. 글 생성 시 체크리스트

투자 유치 보도자료를 생성할 때 아래 항목을 순서대로 확인:

- [ ] 리드에 회사 수식어 + 회사명(영문명) + 투자 금액 + 라운드 포함
- [ ] 리드 투자자 → 신규 투자자 → 기존 투자자 순서로 서술
- [ ] 누적 투자액 언급
- [ ] 투자자(심사역) 코멘트 포함 (간접 인용)
- [ ] 솔루션(매쉬) 설명: CCTV 기반, 분석 항목 나열
- [ ] 주요 고객사 3~4곳 나열
- [ ] 투자금 활용 계획 (고객 확장, AI 고도화, 글로벌 진출)
- [ ] 대표 코멘트로 마무리 (포부/비전)
- [ ] 전체 300~600자, 4~8문단, ~했다 서술체
