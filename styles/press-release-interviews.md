# 인터뷰/기업 프로필 보도자료 스타일 가이드

> 이 문서는 메이아이 관련 인터뷰/기업 프로필 보도 기사 15편을 분석하여 추출한 스타일 패턴이다.
> 보도자료 작성 시 이 가이드를 따라 일관된 톤과 구조를 유지한다.

---

## 1. 보도자료 유형 세분화

인터뷰/기업 프로필 보도자료는 **4가지 서브타입**으로 나뉜다:

| 서브타입 | 비중 | 설명 | 대표 예시 |
|---------|------|------|----------|
| **심층 인터뷰** | ~20% | 대표 인터뷰를 중심으로 기업 비전/전략을 깊이 있게 다룸 | 더벨, 더에이아이 |
| **Q&A형 인터뷰** | ~7% | 질문-답변 형식으로 제품/기술을 설명 | 세계일보 |
| **기업 프로필/성장 스토리** | ~27% | 기업의 성과, 고객 사례, 확장 전략을 종합적으로 서술 | 벤처스퀘어, 시사저널e(UX확대), AVING |
| **뉴스형 소식** | ~46% | 특정 이벤트(글로벌 IR 선정, 전시회 참가 등)를 간결하게 전달 | ZDNet, 뉴시스, 테크M, 아시아경제, 이투데이 등 |

### 서브타입별 작성 지침

- **심층 인터뷰**: 문제 제기 -> 기업의 해법 -> 기술 설명 -> 성과/고객사 -> 향후 전략 순서로 전개. 대표 인용을 3~5회 삽입.
- **Q&A형 인터뷰**: "Q. ~?" / "A. ~" 형식. 질문은 짧고 명확하게, 답변은 2~4문장으로 구성.
- **기업 프로필**: 성과 데이터를 전면에 배치. 숫자 중심의 사실 나열 후 향후 계획으로 마무리.
- **뉴스형 소식**: 첫 문장에 핵심 팩트. 본문은 프로그램/행사 개요 -> 기업 소개 -> 대표 코멘트 순.

## 2. 문체

### 어미 패턴: **~다 체** (뉴스 서술체)

기본 어미는 **"~다"** 종결이다. "~입니다/~합니다"체는 사용하지 않는다.

```
(O) 메이아이는 글로벌 진출을 가속화한다고 밝혔다.
(O) 삼성전자, LG전자, CGV 등이 주요 고객사다.
(O) 프로그램에 참여했으며, 미서부 권역에도 추가 선정되어 이수할 예정이다.
(X) 메이아이는 글로벌 진출을 가속화합니다.
```

### 자주 쓰는 어미 목록

| 어미 | 용례 | 빈도 |
|------|------|------|
| ~다 | "~고객사다", "~솔루션이다" | 가장 높음 |
| ~밝혔다 | "~강화하겠다고 밝혔다" | 대표 인용 마무리에 필수 |
| ~고 있다 | "~협업 중이다", "~제공하고 있다" | 진행형 서술 |
| ~한다 | "~분석한다", "~제시한다" | 기능/역할 설명 |
| ~(으)며 | "~참여했으며, ~선정되어" | 문장 연결 |
| ~예정이다 | "~이수할 예정이다" | 미래 계획 |
| ~전망이다 | "~매출이 가시화될 전망이다" | 미래 예측 |
| ~방침이다 | "~글로벌 역량을 강화할 방침이다" | 기업 계획 |

### 격식 수준

- **중립적 뉴스 문체**. 감탄이나 과장 표현을 쓰지 않는다.
- 형용사 남용 금지: "놀라운", "획기적인", "혁명적인" 등은 사용하지 않는다.
- 단, "독보적인 기술력", "혁신성을 인정받아" 등 **사실에 기반한 수식**은 허용된다.
- 구어체 배제: "~거든요", "~인데요" 등은 사용하지 않는다.

## 3. 글 구조

### 표준 구조 (심층 인터뷰/기업 프로필)

```
1. 리드 (1~2문단): 기업 소개 + 핵심 뉴스훅
2. 본문 섹션 1: 기술/제품 설명
3. 본문 섹션 2: 고객사/성과 사례
4. 본문 섹션 3: 투자/재무 현황
5. 본문 섹션 4: 글로벌 진출/확장 전략
6. 마무리 (1문단): 대표 인용 + 향후 포부
```

### 표준 구조 (뉴스형 소식)

```
1. 리드 (1문장): [기업명]이 [이벤트]에 선정/참가한다고 밝혔다.
2. 이벤트 상세: 프로그램 개요, 구성, 일정
3. 기업 소개: 1~2문장으로 매쉬 솔루션과 고객사 언급
4. 마무리: 대표 인용 1회
```

### 섹션 구분

- **마크다운 H3(###)** 또는 **볼드 소제목**으로 섹션을 명확히 구분한다.
- 소제목은 내용을 구체적으로 요약한다:
  ```
  (O) "삼성·LG·현대차 협업…장비 도입 원활 강점"
  (O) "5년 연속 200% 매출 성장"
  (O) "일본 PoC 시작…시리즈B 라운드 진행 중"
  (X) "주요 성과" (너무 추상적)
  ```

## 4. 리드(첫 문단) 작성법

### 패턴 A: 문제-해법 리드 (심층 인터뷰용)

업계 문제를 먼저 제시하고, 기업이 해결책임을 밝히는 구조.

```
오프라인 매장의 고객 행동 데이터는 대부분 기업에게 여전히 미지의 영역이다. 방문객의
동선, 제품 앞 체류 시간, 미구매 이유 등을 파악할 수단이 없었기 때문이다. [...]
메이아이는 기존 CCTV만으로 매장 방문객의 동선을 실시간 추적하고 구매 전환까지
분석하는, 오프라인용 '구글 애널리틱스'를 표방하는 스타트업이다.
```

### 패턴 B: 팩트 리드 (뉴스형/기업 프로필용)

핵심 사실을 첫 문장에 집약. 가장 빈번하게 사용되는 패턴.

```
영상처리 AI 스타트업 메이아이가 기술보증기금 주관 아기유니콘 육성사업의
'글로벌 IR' 프로그램에 선정되어 글로벌 진출을 가속화한다고 밝혔다.
```

```
영상처리 AI 솔루션 '매쉬(mAsh)'를 개발·제공 중인 스타트업 메이아이(mAy-I)가
독보적인 기술력을 바탕으로 잇따른 성과를 이뤄내고 있다.
```

### 패턴 C: 성과 나열 리드 (성장 스토리용)

연도별 이정표를 간결하게 제시.

```
메이아이는 3년 연속 성장 이정표를 세웠다:
- 2022년: 국제 학술대회 'CVPR' 논문 채택
- 2023년: 삼성벤처투자 주도 시리즈 A 투자(60억 규모) 완료
- 2024년 초: CES 2024 혁신상 수상
```

### 리드 정보 배치 순서

1. **기업 수식어**: "영상처리 AI 스타트업" / "CCTV 영상 기반 AI 스타트업"
2. **기업명**: "메이아이" (또는 "메이아이(mAy-I)")
3. **핵심 뉴스**: 선정/수상/계약/진출 등의 팩트
4. **출처**: "~밝혔다" / "~발표했다"

## 5. 인용문 활용

### 인용 빈도

| 보도자료 유형 | 인용 횟수 |
|----------|----------|
| 심층 인터뷰 | 3~6회 |
| 기업 프로필 | 1~3회 |
| 뉴스형 소식 | 1회 (마무리에 배치) |

### 인용 배치 위치

- **마무리 인용 (필수)**: 거의 모든 보도자료가 대표 코멘트로 마무리된다.
  ```
  박준혁 대표는 "해외 VC와의 네트워크 등 다양한 지원을 적극 활용해
  글로벌 진출 역량을 빠르게 강화할 것"이라고 말했다.
  ```

- **중간 인용**: 기술 설명이나 전략 설명의 근거로 삽입.
  ```
  김찬규 대표는 "오프라인 공간에는 디지털화하지 못한 데이터가 너무 많다"며
  멀티 카메라 기반 AI 분석 기술로 글로벌 리테일 의사결정 방식을 바꾸겠다는
  포부를 밝혔다.
  ```

### 인용문 서술 패턴

인용문 뒤에 붙는 서술 동사 패턴:

```
~(이)라고 밝혔다.        (가장 흔함)
~(이)라고 말했다.        (일반적 발언)
~는 포부를 밝혔다.       (비전/목표 인용 시)
~(이)라고 강조했다.      (중요 메시지 강조)
~(이)라고 덧붙였다.      (추가 코멘트)
~(이)라고 설명했다.      (기술/배경 설명)
~(이)라고 평가했다.      (제3자 평가)
```

### 인용 주체 표기

첫 등장 시 풀네임, 이후 성만 사용:

```
(첫 등장) 김찬규 메이아이 대표는 "..."라고 밝혔다.
(이후)   김 대표는 "..."라고 말했다.
```

또는 직함을 바꿔가며 표기:

```
박준혁 메이아이 대표 → 박준혁 대표 → 박 대표
김찬규 CPO → 김찬규 대표 (직함이 보도자료마다 다를 수 있음)
```

## 6. 숫자/데이터 표기법

### 금액

- 억 단위: **"60억 원"**, **"70억 원"**, **"80억 원"**
- 만 단위: **"약 12억 원"**
- 달러 병기 시: **"80억 원(약 546만 달러)"**
- "시리즈 A 60억 원", "목표 투자금은 약 70억원"

### 퍼센트

- **숫자 + %**: "200%", "92%", "90%", "16%"
- 텍스트 병기 없음 (퍼센트가 아닌 "%" 기호 사용)
- 예: "연평균 200%의 매출 성장", "재계약률 약 90%", "정확도 92%"

### 시간/기간

- **"하루 평균 약 1만9000시간"**, **"2만 5000시간 이상"**
- **"9일간"**, **"5일간"**
- **"3년간 약 12억 원"**
- **"5년 연속"**, **"3년 연속"**

### 날짜

- 연도: "2024년", "2025년"
- 월일: "2025년 1월 15일", "6월 11~14일"
- 분기/반기: "2023년 하반기", "내년 하반기", "올해 하반기~내년 초"

### 매장/수량

- **"약 331개"**, **"101개 매장"**, **"전국 매장 약 80% 이상"**
- **"10개사"**, **"50개 사"**
- **"약 40명 규모"**

### 숫자 강조

- 핵심 수치는 **볼드** 처리: **"시리즈 A 60억 원"**, **"92%"**, **"1000만 시간"**

```

"CCTV 기반 AI 매장 분석 솔루션 기업"
"영상처리 AI 솔루션 '매쉬(mAsh)'를 개발·제공 중인 스타트업 메이아이"
"AI 기반 영상 분석 스타트업 mAy-I Inc."

```

```
"오프라인용 '구글 애널리틱스'"

### 기능/기술 서술 시 반복 표현

```
"기존 CCTV만으로" / "기존 CCTV를 활용해" / "추가 장비·공사 불필요"
"방문객 동선, 성별, 연령대, 체류 시간 등"

```

"유럽 GDPR 및 AI Act 준수"
"잇따른 성과를 이뤄내고 있다"
```

### 고객사 나열 패턴

```
"삼성전자, LG전자, CGV 등"
"삼성전자, LG전자, 현대자동차, 이랜드, CGV 등 국내 대기업"
"삼성전자, LG전자, CGV 등이 주요 고객사다"
"삼성전자, 아모레퍼시픽 등 대기업과 SK쉴더스, 에스원 등 보안업체"
```

- 고객사 언급 시 "등"으로 마무리하여 더 많은 고객이 있음을 암시
- 대기업 3~5개를 구체적으로 나열

### 투자/인증 관련 표현

```
"시리즈 A 투자 유치" / "시리즈 B 라운드 준비(진행) 중"

"TechCrunch Startup Battlefield 200 선정"
"ICLR 2025 스포트라이트 선정"
```

## 8. 회사 소개 패턴

보도자료 내에서 메이아이를 소개할 때 반복되는 **표준 소개 블록**이 있다. 뉴스형 보도자료에서 특히 정형화되어 있다.

### 표준 소개 블록 (1~2문장)

```
메이아이는 영상처리 AI 솔루션 '매쉬(mAsh)'를 개발·운영하며,
CCTV를 활용해 오프라인 매장 방문객 데이터를 분석하고 KPI 인사이트를 제공한다.
삼성전자, LG전자, CGV 등을 고객사로 보유하고 있다.
```

### 확장 소개 블록 (심층 인터뷰용)

```
메이아이는 기존 CCTV만으로 매장 방문객의 동선을 실시간 추적하고 구매 전환까지
분석하는, 오프라인용 '구글 애널리틱스'를 표방하는 스타트업이다.
[...] 방문객 수, 성별·연령대, 전환율 등 고객 경험 정량화, 퍼널 분석,
A/B 테스트 기능을 제공한다.
```

### 소개에 포함되는 요소 (우선순위)

1. 솔루션 이름 ("매쉬" 또는 "mAsh")
2. 핵심 가치 ("CCTV 활용", "추가 장비 불필요")
3. 분석 대상 ("방문객 동선/성별/연령대/전환율")
4. 주요 고객사 (3~5개 나열)
5. 투자/인증 (해당 시)

## 9. 문장 길이와 단락 구성

### 문장 길이

- **평균 30~60자** (한 문장). 뉴스체 특성상 간결하다.
- 복문은 **"~며", "~고", "~어"** 등으로 연결해 최대 80~100자까지 허용.
- 100자 이상 문장은 드물다. 길어지면 두 문장으로 분리.

```
(적절) 메이아이는 아시아 권역에 선발되어 지난 11월 일본·싱가포르에서 9일간
      프로그램에 참여했으며, 미서부 권역에도 추가 선정되어 이수할 예정이다.

(적절) 삼성전자, LG전자, CGV 등이 주요 고객사다.

(너무 김) 메이아이는 영상처리 AI 솔루션 매쉬를 개발하고 운영하면서 오프라인
         공간에 설치된 CCTV를 활용하여 방문객 데이터를 분석하고 매장별 KPI
         및 마케팅 인사이트를 제공하며 삼성전자, LG전자, CGV 등을 고객사로
         보유하고 있는 스타트업이다.
```

### 단락 구성

- **단락당 2~4문장**이 표준.
- 한 단락이 5문장을 넘기지 않는다.
- 주제가 바뀌면 반드시 단락을 나눈다.
- 불릿 포인트(-)를 적극 활용해 정보를 구조화한다.

### 불릿 포인트 활용

정보를 나열할 때 산문보다 **불릿 리스트**를 선호한다:

```
프로그램 구성은 다음과 같다:
- **투자 상담회:** 현지 VC와 1:1 미팅
- **투자 IR:** 글로벌 VC 대상 해외투자 유치
- **네트워킹:** 현지 투자자 및 한인 창업기업과의 교류
```

```
주요 기능:
- 방문객 수, 성별·연령대, 전환율 등 고객 경험 정량화
- 퍼널 분석(병목 지점 파악)
- A/B 테스트 기능
```
