# 데모버전으로 대시보드의 핵심 기능을 체험해보세요

**부제: CCTV 영상의 방문객 데이터, 이렇게 구현돼요**

---

안녕하세요, 오프라인 공간을 더 똑똑하게 만드는 인공지능 스타트업 메이아이입니다.

매쉬를 도입하고자 고민하는 많은 분들이 대시보드의 모습을 궁금해합니다. CCTV 영상을 통해 수집된 방문객 데이터가 시각적으로 어느 수준까지 구현되는지 알고 싶어합니다.

이런 궁금증을 해결하기 위해 메이아이에서는 '데모보드'를 마련했습니다. 매쉬의 데이터 분석 역량을 체험할 수 있도록 샘플 데이터를 활용하여 대시보드를 운영하고 있습니다.

> 샘플 데이터란?: 메이아이 사무실 CCTV 영상을 분석해 데모보드를 만들었습니다. 특정 고객사의 데이터를 활용하지 않았음을 밝힙니다.

하지만 다짜고짜 '이것저것 써보세요' 하면 오히려 제대로 체험하기 어려운 법입니다. 그래서 데모보드 핵심 체험 가이드를 준비했습니다. **오프라인 방문객 데이터 확인에 꼭 필요한 핵심 기능 3가지를 소개합니다.** 메이아이의 고객사가 가장 많이 활용하는 기능으로 엄선했습니다. 이 콘텐츠를 그대로 따라 해보세요.

**들어가기 전에 회원가입부터**
매쉬보드 웹페이지에서 회원가입 절차를 진행해주세요. 3분 안에 가능합니다. 회원가입 후 로그인하면 바로 매쉬보드의 홈 화면을 만날 수 있습니다.

---

## 1. 메이아이 고객사가 매일 확인하는 '매장 방문' 탭

- 홈 화면의 왼쪽 메뉴바에서 **'기본 통계 → 매장 방문'** 탭을 클릭합니다.
- 매장 방문 탭에서는 '방문객 수에 대한 기초 통계'를 확인할 수 있습니다. 일별·시간대별 방문객 추이뿐 아니라, 성별과 연령대에 따른 데이터 분류도 제공합니다.
- 입구를 비추는 CCTV 영상을 통해 방문객의 출입 데이터를 집계하여 대시보드에 표시합니다. 입구가 여러 개일 경우, 각 출입구 CCTV 영상의 데이터를 종합해 그래프 하나에 나타냅니다.
- 페이지 맨 위쪽에서 기간을 설정하면 원하는 기간의 방문객 추이를 확인할 수 있습니다.
- 해당 기능은 메이아이의 모든 요금제 플랜에서 공통으로 제공됩니다.
- "매장 방문 탭 어떻게 활용하나요?" → 매장별 운영 전략을 세분화한 스파오의 고객 사례를 읽어보세요.

---

## 2. 매장 방문객 행동 꿰뚫어보는 '방문객 동선' 탭

- 왼쪽 메뉴바의 **'방문객 동선'** 탭에서 저니맵, 히트맵, 트래픽맵을 차례로 클릭합니다.
- 방문객 동선 탭에서는 '매장 방문객의 동선을 시각화'한 데이터를 제공합니다. 저니맵을 통해서는 매장 방문객의 주요 이동 동선을, 히트맵과 트래픽맵을 통해서는 매장 도면을 바탕으로 주요 체류 구역과 구역 간 이동량을 파악할 수 있습니다.
- 메이아이만의 딥러닝 기술을 통해, 매장 내 서로 다른 CCTV에 나타난 방문객을 하나의 객체로 인식해 분석하고 이를 시각화 데이터로 변환해 대시보드에 표시합니다.
- 각 데이터맵에서 '차트설정', '시각화 옵션' 등을 조작하여 매장 방문객 동선을 확인하기 좋게 커스터마이징 할 수 있습니다.
- "동선 시각화 3종 세트 어떻게 활용하나요?" → 메이아이 Data Analyst의 매장 유동인구 분석 가이드를 읽어보세요.

---

## 3. 우리 매장 맞춤 위젯은 '템플릿 둘러보기'에서!

- 왼쪽 메뉴바에서 **'대시보드'** 탭을 누른 뒤 '템플릿 둘러보기'를 선택하면, 템플릿이 생성됩니다. 템플릿은 고객사의 수요와 업종별 문의 빈도를 기반으로 5종을 마련했습니다.
- 자사 상황에 가장 가까운 템플릿을 선택합니다. 딱 알맞은 템플릿을 찾기 어렵다면 템플릿 5종을 차례로 자유로이 살펴보고 하나를 선택합니다.
- 템플릿을 만들었다면 대시보드 내에서 위젯의 위치를 옮기거나 크기를 조절하는 등 대시보드를 커스터마이징 해봅니다.
- 위젯 내의 '엑셀 복사' 버튼을 눌러 데이터를 엑셀에 기록하거나, '차트 설정' 버튼을 눌러 그래프 모양을 변경할 수 있습니다.
- 우측 상단의 '+위젯 추가하기' 버튼을 눌러 원하는 데이터 위젯을 추가하면, 템플릿을 기반으로 자사 성과 측정에 필요한 데이터셋을 정교하게 마련해나갈 수 있습니다.
- "템플릿 딱 하나 봐야 한다면 뭐가 좋을까요?" → 행사 효과 분석 템플릿을 아모레퍼시픽의 고객사례와 함께 살펴보세요.
