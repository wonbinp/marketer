# 고객사 도입 보도자료 스타일 가이드

> 이 문서는 메이아이의 고객사(코웨이, 스파오, LG유플러스 등) 솔루션 도입/확장 계약을 다루는 보도자료의 작성 스타일을 정의한다. 15편의 실제 보도 기사를 분석하여 도출했다.

---

## 1. 문체

### 어미 패턴
- **기본 어미:** `~했다`, `~밝혔다`, `~전했다`, `~말했다` (과거형 종결)
- **계획/예정:** `~할 예정이다`, `~할 계획이다`, `~할 방침이다`
- **설명/정의:** `~이다`, `~한다` (현재형)
- **인용 도입:** `~라고 밝혔다`, `~라고 말했다`, `~라고 강조했다`, `~라고 전했다`

### 격식 수준
- 경어체가 아닌 **격식체 평서문** (합쇼체/하십시오체가 아닌 해라체)
- `~습니다`는 사용하지 않음
- 감탄사, 의문문, 명령문 없음
- 객관적이고 건조한 보도문 톤

### 예시
```
영상처리 AI 스타트업 메이아이(대표 김찬규)가 코웨이와 매장 분석 AI 솔루션 '매쉬(mAsh)' 확장 도입 계약을 맺고, 브랜드 체험·판매 매장인 '코웨이 갤러리' 전국 26개점에 솔루션을 본격 공급한다고 발표했다.
```

```
메이아이는 사전에 스파오 매장 한 곳에서 기술 검증을 완료했으며, 현재 전국 스파오 매장 80% 이상에 매쉬를 적용했다.
```

## 2. 글 구조

### 전형적인 구성 순서

```
1. 리드 (첫 문단) ── 계약 체결 사실 + 핵심 수치
2. 도입 경과 ─────── PoC → 정식 공급 → 확장의 타임라인
3. 주요 기능 설명 ─── 솔루션이 제공하는 구체적 기능
4. 기술적 특징 ────── 차별화 포인트 (개인정보 보호, 정확도 등)
5. 고객사 활용 방식 ── 고객사가 데이터를 어떻게 활용하는지
6. 대표/관계자 코멘트 ─ 인용문으로 마무리
7. (선택) 보충 설명 ── 고객사 매장/브랜드 소개
```

### 구조적 특징
- 역피라미드 구조: 가장 중요한 정보(계약 체결 사실)가 첫 문장에 위치
- 도입 경과(타임라인)는 거의 모든 보도자료에 포함되며, 불릿 리스트 또는 표로 제시
- 마무리는 거의 항상 대표/관계자 인용문으로 끝남

## 3. 리드(첫 문단) 작성법

### 필수 포함 요소 (등장 순서)
1. **메이아이 소개**: `영상처리 AI 스타트업 메이아이` 또는 `AI 스타트업 메이아이(대표 김찬규)`
2. **고객사명**: `코웨이`, `이랜드월드` 등
3. **계약 유형**: `확장 계약을 체결했다`, `공급 계약을 맺었다`
4. **솔루션명**: `매장 분석 AI 솔루션 '매쉬(mAsh)'`
5. **적용 대상**: `전국 26개점`, `101개 매장` 등 구체적 숫자
6. **발표 동사**: `~라고 밝혔다`, `~라고 발표했다`

### 리드 패턴 예시

**패턴 A: 한 문장형 (가장 빈도 높음)**
```
영상처리 AI 스타트업 메이아이(대표 김찬규)가 코웨이와 매장 분석 AI 솔루션 '매쉬(mAsh)' 확장 도입 계약을 맺고, 브랜드 체험·판매 매장인 '코웨이 갤러리' 전국 26개점에 솔루션을 본격 공급한다고 발표했다.
```

**패턴 B: 두 문장형**
```
영상처리 AI 스타트업 메이아이가 이랜드월드 운영 SPA브랜드 '스파오' 매장에 방문객 데이터 분석 솔루션 '매쉬'를 공급한다고 26일 밝혔다. 이번 계약으로 전국 스파오 매장 101곳에 매쉬를 공급해 방문객의 동선·행동 패턴 등을 예측할 예정이다.
```

**패턴 C: 계약 + 기간 명시형**
```
영상 처리 AI 스타트업 메이아이(대표 김찬규)가 LG유플러스와 기술 검증(PoC) 계약을 체결했다. 오는 5월까지 오프라인 복합문화공간 '일상비일상의틈byU+'에 방문객 분석 AI 솔루션 '매쉬(mAsh)'를 공급한다.
```

### 주의사항
- 리드에서 기술 상세 설명은 하지 않음 (무엇을, 누구에게, 얼마나만 전달)
- 날짜는 `~일 밝혔다` 형태로 자연스럽게 삽입하거나 생략

## 4. 자주 쓰는 표현/관용구

### 계약 관련
- `확장 계약을 체결했다` / `공급 계약을 맺었다`
- `본격 공급한다` / `확장 공급하기로 했다`
- `솔루션을 도입했다` / `서비스를 제공한다`

### 도입 경과 표현
- `PoC(기술 검증)를 진행했다` / `기술 실증을 완료했다`
- `정식 공급을 시작했다` / `정식 도입`
- `~개 매장으로 확대` / `~개 지점으로 확장`
- `하반기 신규 오픈 매장 대상으로 지속 확장 예정`

### 기술/기능 관련
- `자체 개발 딥러닝 알고리즘 기반`
- `핵심성과지표(KPI)와 인사이트를 제공`
- `안면 인식 기술을 사용하지 않아 개인정보 침해 우려가 없다`
- `90% 이상의 정확도로 직원을 식별`
- `98% 정확도로 측정`
- `집계 오류를 최소화`

### 고객 가치 관련
- `매장 운영 퍼포먼스 평가`
- `정확한 객단가 산출`
- `데이터 기반 의사결정 지원`
- `데이터 기반 운영 혁신을 가속화`
- `매출 증대에 실질적으로 기여`
- `공간 경험 개선`
- `맞춤형 판매 전략 강화`

### 마무리 관련
- `~의 실효성을 입증하는 대표적인 사례`
- `지속적인 기술 지원을 이어갈 것`
- `앞장서겠다`

## 5. 인용문 활용

### 배치 위치
- **보도자료 말미** (90% 이상의 보도자료에서 마지막 또는 마지막 직전 단락)
- 일부 보도자료에서 고객사 관계자 인용이 본문 중간에 배치되기도 함

### 인용 주체
- **메이아이 대표** (김찬규 대표 또는 CPO): 거의 모든 보도자료에 등장
- **고객사 관계자**: 일부 보도자료에서 추가 인용 (예: `스파오 빅데이터 팀 관계자`)

### 인용문 도입 패턴
```
김찬규 메이아이 대표는 "코웨이와의 협업은 매장 데이터 분석의 실효성을 입증하는 대표적인 사례"라고 밝혔다.
```
```
김찬규 메이아이 CPO는 "스파오의 데이터 기반 운영 혁신을 가속화하고, 매출 증대에 실질적으로 기여할 수 있도록 지원을 이어가겠다"고 말했다.
```
```
스파오 빅데이터 팀 관계자는 "매쉬를 통해 데이터를 정교하게 분석하면 시장 트렌드를 빠르게 반영할 수 있을 것"이라고 밝혔다.
```

### 인용문 구성
- 직접 인용(`"..."`) + 간접 서술 혼합이 일반적
- 인용문 내용: 협업 의의 + 향후 포부
- 한두 문장으로 간결하게

## 6. 숫자/데이터 표기법

### 매장 수
- 아라비아 숫자 사용: `26개점`, `101개 매장`, `3개 지점`
- `개점`, `개 매장`, `개 지점`, `곳` 혼용

### 퍼센트
- `90% 이상`, `98% 정확도`, `80% 이상`
- `%` 기호 사용 (퍼센트 한글 표기 아님)

### 연도/날짜
- `2023년`, `2024년`, `현재` (연도는 숫자+년)
- `26일 밝혔다` (본문 내 날짜 표기)
- `하반기`, `올해 안으로`

### 타임라인 표기
불릿 리스트 또는 표 형태:
```
- 2023년: 코웨이 갤러리 3개점에서 PoC 진행 → 같은 해 9개점 정식 공급
- 2024년: 18개 매장으로 확대
- 현재: 26개 지점 운영 중
```

또는:

```
| 연도 | 내용 |
|------|------|
| 2023년 | PoC 3개점 → 정식 9개점 |
| 2024년 | 18개점 |
| 2025년 현재 | 26개점 |
```

### 화살표 사용
- `→`를 단계 진행, 인과 관계 표시에 빈번히 사용
- 예: `PoC 진행 → 정식 공급`, `안면 인식 미사용 → 개인정보 침해 우려 없음`

## 7. 회사 소개 패턴

### 메이아이 첫 등장 시 수식어 (택 1)
- `영상처리 AI 스타트업 메이아이`
- `영상처리 AI 전문기업 메이아이`
- `AI 스타트업 메이아이`
- `AI 기반 매장 분석 솔루션 '매쉬'를 운영하는 스타트업 메이아이`

### 대표명 병기
- 첫 등장에서 괄호로 표기: `메이아이(대표 김찬규)` 또는 `메이아이(대표 박준혁)`
- 인용문에서: `김찬규 메이아이 대표`, `김찬규 메이아이 CPO`

### 솔루션 소개 패턴
- 첫 등장: `매장 분석 AI 솔루션 '매쉬(mAsh)'` 또는 `방문객 데이터 분석 솔루션 '매쉬(mAsh)'`
- 이후: `매쉬`로 약칭
- 기술 설명: `자체 개발 딥러닝 알고리즘을 기반으로 오프라인 매장 영상 데이터를 분석해 핵심성과지표(KPI)와 인사이트를 제공한다`

### 고객사 소개 패턴
- 고객사 브랜드/매장에 대한 간략한 설명을 본문 중간 또는 말미에 배치
- 예: `코웨이 갤러리는 비렉스(BEREX) 침대·안마의자 등 슬립·힐링케어 제품과 정수기, 공기청정기, 비데 등을 직접 체험·구매할 수 있는 복합형 오프라인 매장이다.`

## 8. 문장 길이와 단락 구성

### 문장 길이
- **리드 문장:** 50~80자 내외 (한 문장에 많은 정보를 압축)
- **본문 문장:** 30~60자 내외
- 복문 구조가 많음: `~하며`, `~하고`, `~를 통해` 등으로 절을 연결

### 단락 구성
- **단락당 1~3문장** (대부분 1~2문장으로 짧게 끊음)
- 항목별 정보는 불릿 리스트(`-`)로 정리하는 경우가 많음
- 소제목(###)으로 섹션을 구분하는 패턴도 빈번

### 연결 표현
- 문장 간: `이를 통해`, `이번 계약으로`, `특히`, `또한`, `앞서`, `나아가`
- 단락 간: 소제목 또는 구분선(`---`)으로 전환

---

## 9. 전체 글 길이

- **짧은 보도자료:** 약 150~200자 (10~15문장, 요약형)
- **표준 보도자료:** 약 300~500자 (15~25문장)
- **긴 보도자료:** 약 500~700자 (25~35문장, 배경 설명 포함)
- 대부분의 보도자료가 **300~500자** 범위에 분포

## 10. 핵심 작성 원칙 요약

1. **첫 문장에서 승부한다:** 누가(메이아이) + 무엇을(솔루션 공급) + 누구에게(고객사) + 규모(매장 수)를 한 문장에 담는다.
2. **타임라인은 필수다:** PoC에서 확장까지의 단계적 도입 경과를 반드시 포함한다.
3. **기술은 간결하게, 가치는 구체적으로:** 딥러닝 알고리즘 등 기술 설명은 1~2문장, 고객사가 얻는 가치(매장 운영 최적화, 매출 증대 등)는 더 구체적으로 서술한다.
4. **정확도 수치를 강조한다:** `90% 이상`, `98% 정확도` 등 신뢰성을 보여주는 숫자를 볼드 처리한다.
5. **대표 인용문으로 마무리한다:** 협업의 의의 + 향후 포부를 담은 코멘트로 보도자료를 닫는다.
6. **객관적 보도 톤을 유지한다:** 홍보성 과장 표현을 피하고, 사실 중심으로 서술한다.
