# 보도자료 출력 양식

보도자료 작성 결과물은 아래 양식을 따른다. 본문 작성은 유형별 스타일 가이드(`press-release-*.md`)를 따르되, 최종 출력 시 이 양식으로 감싼다.

---

## 출력 구조

docx 파일은 `python-docx`로 **1열 3행 테이블** 형태로 생성한다.

```
┌─────────────────────────────────────┐
│ 1행: 배포 정보 (회색 배경 #F2F2F2)     │
│                                     │
│ 보도자료                             │
│ 희망 배포일: YYYY년 MM월 DD일          │
│ 배포: (주)메이아이                     │
├─────────────────────────────────────┤
│ 2행: 제목 및 내용 (흰 배경)            │
│                                     │
│ [제목] (볼드, 14pt)                   │
│ [부제] (10pt, 회색)                   │
│ [이미지 예시: 설명] (9pt, 이탤릭, 회색) │
│ [본문] (10pt)                        │
│ [인용문] (10pt)                      │
├─────────────────────────────────────┤
│ 3행: 회사 소개 + 문의 (회색 배경)       │
│                                     │
│ 회사 소개 (볼드, 9pt)                 │
│ [보일러플레이트] (9pt, 회색 글씨)       │
│                                     │
│ 문의 (볼드, 9pt)                     │
│ [문의처] (9pt, 회색 글씨)             │
└─────────────────────────────────────┘
```

> **docx 스타일 규칙**:
> - 테이블 테두리: 연한 회색(#BFBFBF), 단선
> - 1행/3행 배경: #F2F2F2 (연한 회색), 2행 배경: 흰색
> - 1행/3행 글씨: 9pt, 회색(#505050)
> - 2행 본문: 10pt, 검정, 줄간격 1.5
> - 제목: 14pt, 볼드
> - 기본 폰트: 맑은 고딕
> - 페이지 여백: 상하 2cm, 좌우 2.5cm

---

## 각 요소 상세

### 배포 정보
- **배포일**: 사용자가 지정하지 않으면 오늘로부터 4주 후 가장 가까운 화~목요일을 기본값으로 제안
- **엠바고**: 사용자가 엠바고를 요청한 경우에만 `※ 엠바고: YYYY년 MM월 DD일 HH:00 이후 보도 가능` 추가
- **배포처**: `(주)메이아이` 고정

### 제목
- 유형별 스타일 가이드의 제목 패턴을 따라 **2~3안** 제안
- 최종 선택은 사용자에게 맡기되, 1안을 추천안으로 표시

### 부제
- 제목을 보완하는 핵심 메시지 한 줄 (선택사항)
- 제목에 정보가 충분히 담긴 경우 생략 가능

### 이미지
- 부제 아래에 `[이미지: 설명]` 플레이스홀더를 1장 배치
- 보도자료 내용에 맞는 이미지를 예시로 추천한다 (예: `[이미지 예시: 매쉬 대시보드 화면]`, `[이미지 예시: CES 2024 혁신상 수상 현장]`)

### 본문
- 유형별 스타일 가이드의 구조와 체크리스트를 따름
- `~했다` 서술체
- **소제목을 사용하지 않는다.** 보도자료는 기사처럼 단락 흐름으로 작성한다.
- **기사체에 어울리지 않는 기호를 사용하지 않는다.** 금지 기호: `→`, `★`, `●`, `▶`, `■`, `※`, `☞`, `◆`, `▷` 등. 단, 회사 소개/문의 영역의 `■`은 예외.
- 상세 작성법은 `press-release-{유형}.md` 참고

### 보일러플레이트 (회사 소개)
매 보도자료 하단에 동일하게 붙는 정형 문구:

```
(주)메이아이는 영상처리 AI 기반 오프라인 방문객 데이터 분석 솔루션 '매쉬(mAsh)'를 개발·운영하는 기업이다. 매장에 설치된 기존 CCTV를 활용해 방문객의 동선, 체류 시간, 성별·연령대 등을 분석하고, 핵심성과지표(KPI)와 데이터 기반 인사이트를 제공한다. 삼성전자, LG전자, CGV 등 국내 주요 기업이 매쉬를 도입하고 있으며, CES 2024 혁신상 수상 및 ICLR 2025 논문 채택 등 글로벌 수준의 기술력을 인정받고 있다.
```

> **고객사 노출 규칙**: 보일러플레이트에 언급하는 고객사는 아래 프로세스를 따른다.
>
> 1. 아래 전체 목록을 사용자에게 보여주고, 현재 고객사가 아닌 곳을 제거하도록 요청한다.
> 2. 남은 목록에서 브랜드 인지도를 고려해 대표적인 3~4곳을 선정하여 보일러플레이트에 삽입한다.
>
> - **기존 기사에서 확인된 고객사 목록**: 삼성전자, LG전자, CGV, 현대자동차, 코웨이, 스파오(이랜드), LG유플러스
>
> 이 목록에 없는 고객사명은 보일러플레이트에 포함하지 않는다. 신규 고객사가 보도된 경우 이 목록을 업데이트한다.

> **참고**: 보일러플레이트는 보도자료 본문과 동일한 `~했다/~이다` 서술체를 사용한다. 고객사 목록, 수상 이력 등은 최신 상태로 유지해야 하며, 변경 시 이 파일을 업데이트한다.

### 문의처

사용자에게 입력받은 정보로 구성한다. 제공되지 않은 항목은 생략한다.

```
(주)메이아이
담당자: [담당자명]
이메일: [이메일]
전화: [전화번호]
```

## 제목 제안 시 출력 형식

제목은 본문과 별도로 먼저 제안한다:

```
### 제목안

1. (추천) 메이아이, 코웨이와 매장 분석 AI 솔루션 '매쉬' 확장 도입 계약 체결
2. 메이아이, 코웨이 갤러리 전국 26개점에 AI 솔루션 '매쉬' 본격 공급
3. 메이아이-코웨이, AI 기반 매장 분석 솔루션 확장 도입 계약

부제안: 전국 코웨이 갤러리 26개점에 매쉬 본격 공급…매장 운영 데이터 분석 강화
```
